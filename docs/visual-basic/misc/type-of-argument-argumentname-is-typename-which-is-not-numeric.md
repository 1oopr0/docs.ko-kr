# <a name="type-of-argument-argumentname-is-typename-which-is-not-numeric"></a>'\<argumentname>' 인수의 형식이 숫자가 아닌 '\<typename>'입니다.
숫자 인수가 필요한 상황에서 숫자가 아닌 인수가 발생합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 인수가 숫자인지 확인합니다.
