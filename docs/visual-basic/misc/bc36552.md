## <a name="example"></a><span data-ttu-id="4c946-110">예제</span><span class="sxs-lookup"><span data-stu-id="4c946-110">Example</span></span>  

 <span data-ttu-id="4c946-111">다음 예제에서 첫 번째 매개 변수는 `Print` 메서드가 `String` 데이터 형식을 확장하는 것을 나타냅니다.</span><span class="sxs-lookup"><span data-stu-id="4c946-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>  
  
```vb

Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="4c946-112">확장 메서드가 다음과 같이 호출되는 경우 메서드의 `str` 매개 변수는 `greeting`를 호출하는 `String` 의 인스턴스인 `Print`에 바인딩됩니다.</span><span class="sxs-lookup"><span data-stu-id="4c946-112">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="4c946-113">컴파일러에서는 `greeting` 을 확장 메서드 `Print`의 인수로 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="4c946-113">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>

```vb  
greeting.Print()  
```
