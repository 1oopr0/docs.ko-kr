# <a name="unable-to-open-key-file-filename-error"></a>' ' 키 파일을 열 수 없습니다 \<filename> . \<error>

이 프로젝트에 대해 지정된 키 파일을 열 수 없습니다.  
  
 **오류 ID:** BC31025  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 키 파일이 있고, 다른 애플리케이션에 의해 잠기지 않았는지 확인합니다.
