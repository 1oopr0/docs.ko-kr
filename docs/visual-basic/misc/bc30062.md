# <a name="me-cannot-be-the-target-of-an-assignment"></a>'Me'는 할당 대상일 수 없습니다.

`Me` 키워드는 항상 코드가 현재 실행 중인 클래스의 특정 인스턴스를 가리킵니다. 이 값은 변경할 수 없습니다.  
  
 **오류 ID:** BC30062  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- `Me`에 값을 할당하는 문을 제거합니다.
