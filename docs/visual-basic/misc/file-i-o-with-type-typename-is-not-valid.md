# <a name="file-io-with-type-typename-is-not-valid"></a>' ' 형식의 파일 i/o \<typename> 가 잘못 되었습니다.

지원되지 않는 형식을 파일에 쓰려고 합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 구조체에서 지원되지 않는 형식이 발생하는 위치를 결정하는지 확인하고 유효한 형식으로 변경합니다.
