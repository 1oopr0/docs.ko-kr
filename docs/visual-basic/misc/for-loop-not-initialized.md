# <a name="for-loop-not-initialized-visual-basic"></a>For 루프가 초기화되지 않았습니다(Visual Basic).

`For...Next` 루프 중간으로 점프했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 루프로의 점프를 제거합니다.
