# <a name="xml-name-expected"></a>XML 이름이 필요합니다.
XML 리터럴 또는 XML 축 속성이 이름 없이 지정되었습니다.  
  
 **오류 ID:** BC31146  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- XML 리터럴 또는 XML 축 속성에 대한 이름을 추가합니다.
