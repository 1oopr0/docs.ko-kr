# <a name="method-methodname-has-no-parameter-named-parametername"></a>' \<methodname> ' 메서드에 ' ' (이) 라는 매개 변수가 없습니다. \<parametername>

제공한 명명된 인수가 대상 메서드의 인수와 일치하지 않습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 명명된 인수가 올바르게 제공되었는지 확인합니다.
