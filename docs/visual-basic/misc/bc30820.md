# <a name="lsetstmt-is-not-declared"></a>' \<lsetstmt> '이 선언 되지 않았습니다.

'\<lsetstmt>'이(가) 선언되지 않았습니다. LSet 문은 더 이상 지원되지 않습니다. 대신 Microsoft.VisualBasic.LSet을 사용합니다.  
  
 이전 버전에서 Visual Basic에 내장 된 여러 함수가 <xref:Microsoft.VisualBasic?displayProperty=nameWithType> 네임 스페이스로 이동 되었습니다. 그 결과 모든 프로그래밍 언어에서 더 일반적으로 기능을 사용할 수 있게 되었습니다.  
  
 **오류 ID:** BC30820  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 대신 `LSet` 네임스페이스의 `Microsoft.VisualBasic` 함수를 사용합니다.
