# <a name="end-set-must-be-the-first-statement-on-a-line"></a>'End Set'은 줄의 첫째 문이어야 합니다.
`End Set` 문 앞에 콜론(:) 문 구분 기호가 있습니다. `End Set` 은 해당 소스 줄에서 유일한 문이어야 합니다.  
  
 **오류 ID:** BC32034  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 각 문을 서로 다른 줄로 분리합니다.
