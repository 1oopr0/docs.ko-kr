# <a name="typename-is-an-unsupported-type"></a>' \<typename> '은 지원 되지 않는 형식입니다.

지정 된 형식을 Visual Studio에서 표시할 수 없습니다.  
  
 **오류 ID:** BC30649  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 지원되는 형식을 사용합니다.  
  
- 새 형식을 개발합니다.
