# <a name="class-not-registered-on-local-machine"></a>클래스가 로컬 컴퓨터에 등록되지 않았습니다.

이 오류는 일반적으로 손상된 파일로 인해 발생합니다. Visual Studio를 다시 설치 해야 할 수 있습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- Visual Studio를 다시 설치합니다.
