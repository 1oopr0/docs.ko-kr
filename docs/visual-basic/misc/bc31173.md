# <a name="missing-required-white-space"></a>필요한 공백이 없습니다.
필요한 구분 기호 공백 없이 XML 리터럴을 지정했습니다. 필요한 구분 기호 공간의 예로 XML 요소 리터럴에서 후속 특성 사이의 공백이 있습니다.  
  
 **오류 ID:** BC31173  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 필요한 공백 구분 기호를 추가합니다.
