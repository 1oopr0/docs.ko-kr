# <a name="initialization-error"></a>초기화 오류
프로그램을 초기화하는 동안 오류가 발생했습니다. 진행 중인 설치를 취소한 경우 이러한 오류가 발생할 수 있습니다.  
  
 **오류 ID:** BC2000  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 설치를 다시 시도합니다.
