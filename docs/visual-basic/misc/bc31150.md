# <a name="end-tag-tagname-expected"></a>끝 태그가 \</tagName> 필요 합니다.

XML 요소 리터럴의 끝 태그가 해당 시작 태그와 일치하지 않습니다.  
  
 **오류 ID:** BC31150  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 끝 XML 태그를 수정하여 XML 요소 리터럴의 시작 XML 태그와 일치시킵니다.
