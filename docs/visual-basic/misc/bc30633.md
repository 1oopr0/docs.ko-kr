# <a name="set-statement-must-end-with-a-matching-end-set"></a>'Set' 문은 짝이 되는 'End Set'으로 끝나야 합니다.
`Set` 속성 프로시저는 `End Set` 문으로 끝나야 합니다.  
  
 **오류 ID:** BC30633  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- `Set` 속성 프로시저가 `End Set` 구문으로 끝나는지 확인합니다.
