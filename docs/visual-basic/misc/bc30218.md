# <a name="lib-expected"></a>'Lib'가 필요합니다.

`Declare` 문은 `Lib` 키워드를 사용하지 않고 라이브러리 이름을 지정합니다.  
  
 **오류 ID:** BC30218  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 라이브러리 이름 앞에 `Lib` 키워드를 추가합니다.
