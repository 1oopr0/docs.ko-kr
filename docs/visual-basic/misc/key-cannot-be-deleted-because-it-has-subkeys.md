# <a name="key-cannot-be-deleted-because-it-has-subkeys"></a>하위 키가 있으므로 키를 삭제할 수 없습니다.

하위 키가 있으므로 키를 삭제할 수 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 이 작업을 시도하기 전에 지정된 키의 하위 키를 확인하고 삭제합니다.
