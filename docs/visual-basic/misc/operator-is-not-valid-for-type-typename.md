# <a name="operator-is-not-valid-for-type-typename"></a>' ' 형식에 대 한 연산자가 잘못 되었습니다. \<typename>

연산자가 형식에 적합하지 않습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 연산자가 작업 중인 형식에 적합한지 확인합니다.
