# <a name="the-file-is-currently-closed"></a>파일이 현재 닫혀 있습니다.

파일이 현재 닫혀 있는데 열려 있는 파일에서만 가능한 작업을 수행하려고 했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 파일을 열고 작업을 다시 시도합니다.
