# <a name="types-are-not-available-in-this-context"></a>이 컨텍스트에서는 형식을 사용할 수 없습니다.
`#If` 지시문 내에서 형식을 참조하려고 했습니다.  
  
 **오류 ID:** BC30458  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 참조를 제거합니다.
