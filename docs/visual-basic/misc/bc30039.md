# <a name="loop-control-variable-cannot-be-a-property-or-a-late-bound-indexed-array"></a>루프 제어 변수는 속성 또는 런타임에 바인딩된 인덱스 배열일 수 없습니다.
`For` 루프를 반복하는 데 사용할 수 있는 변수는 숫자 데이터 형식이어야 합니다.  
  
 **오류 ID:** BC30039  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- `Integer`, `Short`, `Long`, `Byte`, `Single`, `Double`또는 `Decimal`을 지정하여 루프 제어 변수 선언을 변경하세요.
