# <a name="interface-can-inherit-only-from-another-interface"></a>인터페이스는 다른 인터페이스에서만 상속할 수 있습니다.
인터페이스가 아닌 항목에서 상속하려고 했습니다.  
  
 **오류 ID:** BC30354  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 상속하려는 기본 인터페이스가 실제로 인터페이스인지 확인합니다.
