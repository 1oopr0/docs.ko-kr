# <a name="option-optionname-requires-argument"></a>\<optionname> 옵션을 사용하려면 가 필요합니다.\<argument>

 **오류 ID:** BC2006  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 누락된 인수를 제공합니다.
