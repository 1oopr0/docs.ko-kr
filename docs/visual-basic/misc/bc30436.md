# <a name="members-in-a-module-cannot-be-declared-specifier"></a>모듈의 멤버는 ' '로 선언할 수 없습니다. \<specifier>

`Module` 문 내에서 멤버에 유효하지 않은 지정자를 사용했습니다. 모듈은 인스턴스화될 수 없고 상속을 지원하지 않으며 인터페이스를 구현할 수 없습니다.  
  
 **오류 ID:** BC30436  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 지정자를 제거합니다.
