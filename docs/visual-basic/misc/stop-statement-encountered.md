# <a name="stop-statement-encountered"></a>Stop 문을 발견했습니다.

실행을 중지하는 `Stop` 문을 발견했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. `Stop` 문을 제거합니다.
