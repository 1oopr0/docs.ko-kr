# <a name="labels-are-not-valid-in-the-immediate-window"></a>직접 실행 창에서는 레이블을 사용할 수 없습니다.
선 레이블은 분기하도록 설정하며 디버깅 컨텍스트에서는 허용되지 않습니다.  
  
 **오류 ID:** BC30134  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- **직접 실행** 창에서 선 레이블을 입력하지 마세요.
