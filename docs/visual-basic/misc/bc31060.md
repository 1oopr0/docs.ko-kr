# <a name="propertyname-implicitly-defines-membername-which-conflicts-with-a-member-of-the-same-name-in-type-typename"></a>' '은 ' ' \<propertyname> \<membername> 에서 동일한 이름의 멤버와 충돌 하는 ' '을 암시적으로 정의 \<type> 합니다. \<typename>
형식 멤버의 이름이 속성에 대해 암시적으로 만든 멤버 이름과 충돌합니다. 속성은 정의된 멤버와 충돌할 수 있는 여러 암시적 변수를 암시적으로 만듭니다.  
  
 **오류 ID:** BC31060  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 이름 충돌을 제거하려면 명시적으로 선언된 멤버 이름을 바꿉니다.
