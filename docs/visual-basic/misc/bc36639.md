<span data-ttu-id="e5011-103">`Sub` 또는 함수에서 선언된 람다 식이 해당 `ByRef` 또는 함수의 `Sub` 매개 변수를 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="e5011-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="e5011-104">예를 들어 다음 코드는 `ByRef` 매개 변수 `n` 이 람다 식에 사용되기 때문에 이 오류가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="e5011-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>

