# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="bea60-102">람다 식에는 'ByRef' 매개 변수 '\<parametername>'을 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="bea60-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>

<span data-ttu-id="bea60-103">`Sub` 또는 함수에서 선언된 람다 식이 해당 `ByRef` 또는 함수의 `Sub` 매개 변수를 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="bea60-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="bea60-104">예를 들어 다음 코드는 `ByRef` 매개 변수 `n` 이 람다 식에 사용되기 때문에 이 오류가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="bea60-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```vb  
'' Not valid.
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="bea60-105">**오류 ID:** BC36639</span><span class="sxs-lookup"><span data-stu-id="bea60-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="bea60-106">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="bea60-106">To correct this error</span></span>  
  
- <span data-ttu-id="bea60-107">다음 코드와 같이 `ByRef` 매개 변수를 지역 변수에 할당하고 람다 식에서 지역 변수를 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="bea60-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```vb  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
