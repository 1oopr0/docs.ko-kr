<span data-ttu-id="c8281-103">`Structure` 제약 조건이 있는 형식 매개 변수가 다른 형식 매개 변수에 대한 제약 조건으로 사용되었습니다.</span><span class="sxs-lookup"><span data-stu-id="c8281-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="c8281-104">`Structure` 제약 조건에서는 해당 형식 매개 변수에 전달된 형식 인수가 값 형식이어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="c8281-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="c8281-105">그러나 값 형식을 구현하거나 상속할 수 없으므로 제약 조건으로 사용해도 의미가 없으며, 다른 형식 매개 변수가 구현하거나 상속해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="c8281-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>

 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="c8281-109">`t2` 에 전달된 형식은 값 형식이어야 하므로 `t1`에 전달된 형식이 `t1` 에 전달된 형식을 구현하거나 상속할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="c8281-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>
