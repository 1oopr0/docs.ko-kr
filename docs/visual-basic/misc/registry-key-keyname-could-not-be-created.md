# <a name="registry-key-keyname-could-not-be-created"></a>'\<keyname>' 레지스트리 키를 만들 수 없습니다.
지정된 레지스트리 키를 만들 수 없습니다. 보안 권한이 없기 때문일 수 있습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 키에 액세스할 수 있는 보안 권한이 있는지 확인합니다.
