# <a name="property-value-is-not-valid"></a>속성 값이 잘못되었습니다.

개체 또는 컨트롤의 속성 중 하나가 범위를 벗어난 값으로 설정되었습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 속성 값을 유효한 설정으로 변경합니다.
