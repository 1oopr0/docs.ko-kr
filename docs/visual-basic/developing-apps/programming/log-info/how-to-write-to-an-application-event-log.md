    1. <span data-ttu-id="8f9bd-113">**프로젝트** 메뉴에서 **새 항목 추가**를 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="8f9bd-113">On the **Project** menu, choose **Add New Item**.</span></span>
