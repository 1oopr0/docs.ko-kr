# <a name="byref-visual-basic"></a>ByRef(Visual Basic)

호출 된 프로시저가 호출 코드에서 인수를 기반으로 하는 변수의 값을 변경할 수 있는 방식으로 인수가 전달 되도록 지정 합니다.  
  
## <a name="remarks"></a>설명  

 `ByRef` 한정자는 다음 컨텍스트에서 사용할 수 있습니다.

 [Function 문](../statements/function-statement.md)
