# <a name="overflow-visual-basic-error"></a>오버플로가 발생했습니다(Visual Basic 오류).

리터럴은 할당 되는 데이터 형식의 제한을 벗어난 값을 나타냅니다.  
  
 **오류 ID:** BC30036  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 대상 데이터 형식에 대 한 값 범위를 확인 하 고 해당 범위를 따르도록 리터럴을 다시 작성 하세요.
