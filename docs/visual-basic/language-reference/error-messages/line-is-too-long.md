# <a name="line-is-too-long"></a>줄이 너무 깁니다.
소스 텍스트 줄은 65535 자를 초과할 수 없습니다.  
  
 **오류 ID:** BC30494  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 줄의 길이를 65535 자 이하로 줄이십시오.
