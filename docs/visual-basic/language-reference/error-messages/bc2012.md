# <a name="cant-open-filename-for-writing"></a>'\<filename>'을(를) 쓰기용으로 열 수 없습니다.
지정 된 파일이 이미 열려 있기 때문에 쓰기용으로 열 수 없습니다.  
  
 **오류 ID:** BC2012  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 파일을 닫고 다시 엽니다.  
  
2. 파일의 사용 권한을 확인 합니다.
