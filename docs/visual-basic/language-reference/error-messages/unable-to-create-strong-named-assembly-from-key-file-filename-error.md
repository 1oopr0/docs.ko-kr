# <a name="unable-to-create-strong-named-assembly-from-key-file-filename-error"></a>'\<filename>' 키 파일에서 강력한 이름의 어셈블리를 만들 수 없습니다. \<error>

지정 된 키 파일에서 강력한 이름의 어셈블리를 만들 수 없습니다.  
  
 **오류 ID:** BC31026  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 올바른 키 파일을 지정 했 고 다른 응용 프로그램에 의해 잠겨 있지 않은지 확인 하십시오.
