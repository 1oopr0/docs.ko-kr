# <a name="optional-expected"></a>'Optional'이 필요합니다.

프로시저 선언에서 선택적 인수 뒤에는 필수 인수가 있습니다. 선택적 인수 다음에 나오는 모든 인수는 선택 사항 이어야 합니다.  
  
 **오류 ID:** BC30202  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 인수가 필요한 경우 인수 목록의 첫 번째 선택적 인수 앞에 오도록 이동 합니다.  
  
2. 인수를 선택적으로 사용 하려면 키워드를 사용 `Optional` 합니다.
