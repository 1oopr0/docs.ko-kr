# <a name="name1-is-ambiguous-imported-from-the-namespaces-or-types-name2"></a>'\<name1>' 네임스페이스 또는 형식에서 가져온 '\<name2>'은(는) 모호합니다.

모호한 이름을 제공했으므로 다른 이름과 충돌합니다. Visual Basic 컴파일러에 충돌 해결 규칙이 없습니다. 이름을 명확 하 게 구분 해야 합니다.  
  
 **오류 ID:** BC30561  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 네임 스페이스 가져오기를 제거 하 여 이름을 명확 하 게 구분 합니다.  
  
2. 이름을 정규화합니다.
