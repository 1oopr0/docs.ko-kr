# <a name="compiler-error-cs1672"></a>컴파일러 오류 CS1672

/platform에 대해 잘못된 'option' 옵션입니다. anycpu, x86, Itanium 또는 x64여야 합니다.  
  
 이 옵션은 프로세서 형식을 지정합니다. 나열된 형식 중 하나를 사용합니다.
