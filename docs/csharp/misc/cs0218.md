# <a name="compiler-error-cs0218"></a><span data-ttu-id="18555-103">컴파일러 오류 CS0218</span><span class="sxs-lookup"><span data-stu-id="18555-103">Compiler Error CS0218</span></span>

<span data-ttu-id="18555-104">'type' 형식에는 true 및 false 연산자 선언이 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="18555-104">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
<span data-ttu-id="18555-105">사용자 정의 형식이 [& operator](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) 또는 [&#124; 연산자](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-)를 오버 로드 하는 경우에는 단락 [&& 연산자나](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) [&#124;&#124; 연산자](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) 를 정의 하기 위해 [true 및 false](../language-reference/operators/true-false-operators.md) 연산자도 정의 해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="18555-105">If a user-defined type overloads the [& operator](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) or [&#124; operator](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), it must also define [true and false](../language-reference/operators/true-false-operators.md) operators, in order to make short-circuiting [&& operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) or [&#124;&#124; operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) defined.</span></span>
  
 <span data-ttu-id="18555-106">다음 샘플에서는 CS0218을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="18555-106">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
