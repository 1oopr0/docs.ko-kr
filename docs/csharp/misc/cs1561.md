# <a name="compiler-error-cs1561"></a>컴파일러 오류 CS1561

출력 파일 이름이 너무 길거나 잘못되었습니다.  
  
 출력 파일 이름은 256자보다 길 수 없으며 유로 기호, 물음표 또는 백슬래시 등의 문자를 포함할 수 없습니다.
