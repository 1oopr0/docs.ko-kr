# <a name="compiler-error-cs0557"></a><span data-ttu-id="e0c1f-103">컴파일러 오류 CS0557</span><span class="sxs-lookup"><span data-stu-id="e0c1f-103">Compiler Error CS0557</span></span>
<span data-ttu-id="e0c1f-104">'class' 형식의 사용자 정의 변환이 중복되었습니다.</span><span class="sxs-lookup"><span data-stu-id="e0c1f-104">Duplicate user-defined conversion in type 'class'</span></span>  
  
 <span data-ttu-id="e0c1f-105">중복 변환 루틴은 클래스에서 허용되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="e0c1f-105">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="e0c1f-106">다음 예제에서는 CS0557을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="e0c1f-106">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
