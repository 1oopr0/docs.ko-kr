<span data-ttu-id="c284e-104">읽기 전용 필드 'identifier'의 멤버에는 ref 또는 out을 전달할 수 없습니다. 단 생성자에서는 예외입니다.</span><span class="sxs-lookup"><span data-stu-id="c284e-104">Members of readonly field 'identifier' cannot be passed ref or out (except in a constructor)</span></span>  
  
 <span data-ttu-id="c284e-105">이 오류는 `readonly` 필드의 멤버인 함수에 변수를 `ref` 또는 `out` 인수로 전달하는 경우에 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="c284e-105">This error occurs if you pass a variable to a function that is a member of a `readonly` field as a `ref` or `out` argument.</span></span> <span data-ttu-id="c284e-106">`ref` 및 `out` 매개 변수를 수정할 수 있기 때문에 이 작업은 허용되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="c284e-106">Since `ref` and `out` parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="c284e-107">이 오류를 해결하려면 필드의 `readonly` 키워드를 제거하거나 `readonly` 필드의 멤버를 함수에 전달하지 마세요.</span><span class="sxs-lookup"><span data-stu-id="c284e-107">To resolve this error, remove the `readonly` keyword on the field, or do not pass the members of the `readonly` field to the function.</span></span> <span data-ttu-id="c284e-108">예를 들어 다음 예제와 같이 수정할 수 있는 임시 변수를 만들고 임시 변수를 인수로 전달할 수 있습니다 `ref` .</span><span class="sxs-lookup"><span data-stu-id="c284e-108">For example, you might try creating a temporary variable which can be modified and passing the temporary as a `ref` argument, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c284e-109">예제</span><span class="sxs-lookup"><span data-stu-id="c284e-109">Example</span></span>  
 <span data-ttu-id="c284e-110">다음 샘플에서는 CS1649를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="c284e-110">The following sample generates CS1649:</span></span>  
  
```csharp  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
