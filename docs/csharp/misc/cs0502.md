 <span data-ttu-id="4e775-105">클래스 멤버는 [abstract](../language-reference/keywords/abstract.md) 이면서 [sealed](../language-reference/keywords/sealed.md)일 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="4e775-105">A class member cannot be both [abstract](../language-reference/keywords/abstract.md) and [sealed](../language-reference/keywords/sealed.md).</span></span>  
  
 <span data-ttu-id="4e775-106">다음 샘플에서는 CS0502를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="4e775-106">The following sample generates CS0502:</span></span>  
  
```csharp  
// CS0502.cs

{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
