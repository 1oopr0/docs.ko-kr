# <a name="compiler-warning-level-2-cs0253"></a><span data-ttu-id="e5b08-103">컴파일러 경고(수준 2) CS0253</span><span class="sxs-lookup"><span data-stu-id="e5b08-103">Compiler Warning (level 2) CS0253</span></span>
<span data-ttu-id="e5b08-104">의도하지 않은 참조 비교가 있을 수 있습니다. 값 비교를 가져오려면 오른쪽을 'type' 형식으로 캐스팅하세요.</span><span class="sxs-lookup"><span data-stu-id="e5b08-104">Possible unintended reference comparison; to get a value comparison, cast the right hand side to type 'type'</span></span>  
  
 <span data-ttu-id="e5b08-105">컴파일러가 참조 비교를 수행합니다.</span><span class="sxs-lookup"><span data-stu-id="e5b08-105">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="e5b08-106">문자열의 값을 비교하려는 경우 식의 오른쪽을 `type`로 캐스팅합니다.</span><span class="sxs-lookup"><span data-stu-id="e5b08-106">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="e5b08-107">다음 샘플에서는 CS0253을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="e5b08-107">The following sample generates CS0253:</span></span>  
  
```csharp
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
