# <a name="compiler-error-cs0031"></a><span data-ttu-id="88fb6-103">컴파일러 오류 CS0031</span><span class="sxs-lookup"><span data-stu-id="88fb6-103">Compiler Error CS0031</span></span>

<span data-ttu-id="88fb6-104">'value' 상수 값을 'type'으로 변환할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="88fb6-104">Constant value 'value' cannot be converted to a 'type'.</span></span>

<span data-ttu-id="88fb6-105">해당 형식이 값을 저장할 수 없는 변수에 값을 할당하려고 시도했습니다.</span><span class="sxs-lookup"><span data-stu-id="88fb6-105">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="88fb6-106">자세한 내용은 [형식](../programming-guide/types/index.md)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="88fb6-106">For more information, see [Types](../programming-guide/types/index.md).</span></span>

<span data-ttu-id="88fb6-107">다음 샘플에서는 Checked와 Unchecked 컨텍스트 둘 다에서 CS0031을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="88fb6-107">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>

```csharp
// CS0031.cs
namespace CS0031
{
    public class Program
    {
        public static void Main()
        {
            int num = (int)2147483648M; //CS0031
            // Try using a larger numeric type instead.
            // long num = (long)2147483648M; //CS0031

            const decimal d = -10M; // Decimal literal
            unchecked
            {
                const byte b = (byte)d; // CS0031
                // For small values try using a signed byte instead.
                // const sbyte b = (sbyte)d;
            }
        }
    }
}
```
