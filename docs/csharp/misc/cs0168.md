# <a name="compiler-warning-level-3-cs0168"></a><span data-ttu-id="c7c16-103">컴파일러 경고(수준 3) CS0168</span><span class="sxs-lookup"><span data-stu-id="c7c16-103">Compiler Warning (level 3) CS0168</span></span>
<span data-ttu-id="c7c16-104">' Var ' 변수가 선언 되었지만 사용 되지 않았습니다.</span><span class="sxs-lookup"><span data-stu-id="c7c16-104">The variable 'var' is declared but never used</span></span>

<span data-ttu-id="c7c16-105">변수를 선언할 때 컴파일러에서 수준 3 경고를 실행 하지만 사용 하지는 않습니다.</span><span class="sxs-lookup"><span data-stu-id="c7c16-105">The compiler issues a level-three warning when you declare a variable, but do not use it.</span></span>
  
<span data-ttu-id="c7c16-106">다음 샘플에서는 CS0168 경고를 생성 합니다.</span><span class="sxs-lookup"><span data-stu-id="c7c16-106">The following sample generates a CS0168 warning:</span></span>  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
