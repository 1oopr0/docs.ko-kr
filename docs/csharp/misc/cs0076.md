# <a name="compiler-error-cs0076"></a>컴파일러 오류 CS0076

'value__' 열거자 이름은 예약된 것이므로 사용할 수 없습니다.  
  
 [열거형](../language-reference/builtin-types/enum.md) 항목에 **value__** 라는 식별자가 있어서는 안 됩니다.
