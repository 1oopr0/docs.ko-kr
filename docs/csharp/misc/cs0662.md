# <a name="compiler-error-cs0662"></a><span data-ttu-id="d62dc-103">컴파일러 오류 CS0662</span><span class="sxs-lookup"><span data-stu-id="d62dc-103">Compiler Error CS0662</span></span>

<span data-ttu-id="d62dc-104">'method'는 ref 매개 변수에 Out 특성만 지정할 수는 없습니다.</span><span class="sxs-lookup"><span data-stu-id="d62dc-104">'method' cannot specify only Out attribute on a ref parameter.</span></span> <span data-ttu-id="d62dc-105">In 및 Out 특성을 모두 사용하거나 둘 다 사용하지 마세요.</span><span class="sxs-lookup"><span data-stu-id="d62dc-105">Use both In and Out attributes, or neither.</span></span>  
  
 <span data-ttu-id="d62dc-106">인터페이스 메서드에 [ref](../language-reference/keywords/ref.md) 에서 [Out](xref:System.Runtime.InteropServices.OutAttribute) 특성만 사용하는 매개 변수가 있습니다.</span><span class="sxs-lookup"><span data-stu-id="d62dc-106">An interface method has a parameter that uses [ref](../language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="d62dc-107">`ref` Out **특성을 사용하는** 매개 변수는 [In](xref:System.Runtime.InteropServices.InAttribute) 특성도 사용해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="d62dc-107">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>  
  
 <span data-ttu-id="d62dc-108">다음 샘플에서는 CS0662를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d62dc-108">The following sample generates CS0662:</span></span>  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
