 <span data-ttu-id="e01d9-105">[정적](../language-reference/keywords/static.md) 생성자를 하나 이상의 매개 변수와 함께 선언할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="e01d9-105">A [static](../language-reference/keywords/static.md) constructor cannot be declared with one or more parameters.</span></span> <span data-ttu-id="e01d9-106">자세한 내용은 [생성자](../programming-guide/classes-and-structs/constructors.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="e01d9-106">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="e01d9-107">다음 샘플에서는 CS0132를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="e01d9-107">The following sample generates CS0132:</span></span>  
  
```csharp  
// CS0132.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass  
   {  
      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
