# <a name="compiler-error-cs0228"></a>컴파일러 오류 CS0228

'type'에 'member'에 대한 정의가 없거나 액세스할 수 없습니다.  
  
 **System.Hashtable**, **System.String**또는 **System.Array**의 시스템 버전을 바꿨는데 해당 교체 버전에 `member`가 포함되지 않았을 수 있습니다.  
  
 그렇지 않은 경우 Visual Studio를 복구하거나 다시 설치해야 할 수 있습니다.
