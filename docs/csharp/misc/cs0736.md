## <a name="example"></a><span data-ttu-id="5248c-111">예제</span><span class="sxs-lookup"><span data-stu-id="5248c-111">Example</span></span>  
 <span data-ttu-id="5248c-112">`Program.testMethod` 가 static으로 선언되었기 때문에 다음 코드에서는 CS0736을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="5248c-112">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>

```csharp  
{
  
    interface ITest  
    {  
        int testMethod(int x);  
    }

    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }

    }
}  
```
