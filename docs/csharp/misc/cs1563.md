# <a name="compiler-error-cs1563"></a>컴파일러 오류 CS1563

'output file' 출력에 소스 파일이 없습니다.  
  
 [-Out](../language-reference/compiler-options/out-compiler-option.md) 컴파일러 옵션을 지정 했지만 소스 코드 파일이 없습니다. **/out**을 사용하지 않거나, **/out**뒤에 소스 코드 파일을 지정해야 합니다.
