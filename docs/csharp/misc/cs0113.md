# <a name="compiler-error-cs0113"></a><span data-ttu-id="d5c96-103">컴파일러 오류 CS0113</span><span class="sxs-lookup"><span data-stu-id="d5c96-103">Compiler Error CS0113</span></span>

<span data-ttu-id="d5c96-104">override로 표시된 'function' 멤버는 new 또는 virtual로 표시할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="d5c96-104">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="d5c96-105">[new](../language-reference/keywords/new-modifier.md) 및 [override](../language-reference/keywords/override.md) 키워드로 메서드를 표시할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="d5c96-105">It is mutually exclusive to mark a method with the [new](../language-reference/keywords/new-modifier.md) and [override](../language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="d5c96-106">다음 샘플에서는 CS0113을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d5c96-106">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
