# <a name="compiler-error-cs0711"></a><span data-ttu-id="f583f-103">컴파일러 오류 CS0711</span><span class="sxs-lookup"><span data-stu-id="f583f-103">Compiler Error CS0711</span></span>

 <span data-ttu-id="f583f-105">정적 클래스는 인스턴스화할 수 없으므로 생성자 또는 소멸자가 필요하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="f583f-105">A static class cannot be instantiated, hence it has no need for constructors or destructors.</span></span> <span data-ttu-id="f583f-106">이 오류를 방지하려면 정적 클래스에서 모든 소멸자를 제거하거나, 인스턴스를 생성 및 소멸하려는 경우 클래스를 비정적으로 설정합니다.</span><span class="sxs-lookup"><span data-stu-id="f583f-106">To avoid this error, remove any destructors from static classes, or, if you really want to construct and destroy instances, make the class non-static.</span></span>

```csharp

   ~C()  // CS0711  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
