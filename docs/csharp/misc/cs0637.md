# <a name="compiler-error-cs0637"></a><span data-ttu-id="338a5-103">컴파일러 오류 CS0637</span><span class="sxs-lookup"><span data-stu-id="338a5-103">Compiler Error CS0637</span></span>

<span data-ttu-id="338a5-104">Static 또는 const 필드에는 FieldOffset 특성을 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="338a5-104">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="338a5-105">[FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) 특성은 [정적](../language-reference/keywords/static.md) 또는 [const](../language-reference/keywords/const.md)로 표시된 필드에 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="338a5-105">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../language-reference/keywords/static.md) or [const](../language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="338a5-106">다음 샘플에서는 CS0637을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="338a5-106">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
