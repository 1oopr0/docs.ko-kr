# <a name="compiler-error-cs0272"></a>컴파일러 오류 CS0272
set 접근자에 액세스할 수 없으므로 'property/indexer' 속성 또는 인덱서는 이 컨텍스트에서 사용할 수 없습니다.  
  
 이 오류는 `set` 접근자가 프로그램 코드에 액세스할 수 없을 때 발생합니다. 이 오류를 해결하려면 접근자의 접근성을 향상하거나 호출 위치를 변경합니다. 자세한 내용은 [접근자 액세스 가능성 제한](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md)을 참조하세요.  
  
## <a name="example"></a>예제  
 다음 예제는 CS0272 오류가 발생하는 경우입니다.  
  
```csharp  
// CS0272.cs  
public class MyClass  
{  
    public int Property  
    {  
        get { return 0; }  
        private set { }  
    }  
}  
  
public class Test  
{  
    static void Main()  
    {  
        MyClass c = new MyClass();  
        c.Property = 10;      // CS0272  
        // To resolve, remove the previous line
        // or use an appropriate modifier on the set accessor.  
    }  
}  
```
