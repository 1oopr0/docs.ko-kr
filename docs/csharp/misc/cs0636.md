# <a name="compiler-error-cs0636"></a><span data-ttu-id="1bdf1-103">컴파일러 오류 CS0636</span><span class="sxs-lookup"><span data-stu-id="1bdf1-103">Compiler Error CS0636</span></span>
<span data-ttu-id="1bdf1-104">FieldOffset 특성은 StructLayout(LayoutKind.Explicit)으로 표시된 형식의 멤버에만 배치할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="1bdf1-104">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="1bdf1-105">**FieldOffset** 특성으로 표시된 멤버를 포함하는 경우 구조체 선언의 **StructLayout(LayoutKind.Explicit)** 특성을 사용해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="1bdf1-105">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="1bdf1-106">자세한 내용은 [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="1bdf1-106">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="1bdf1-107">다음 샘플에서는 CS0636을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="1bdf1-107">The following sample generates CS0636:</span></span>  
  
```csharp  
// CS0636.cs  
using System;  
using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]  
struct Worksheet  
{  
   [FieldOffset(4)]public int i;   // CS0636
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
