# <a name="compiler-error-cs0629"></a><span data-ttu-id="a0783-103">컴파일러 오류 CS0629</span><span class="sxs-lookup"><span data-stu-id="a0783-103">Compiler error CS0629</span></span>

<span data-ttu-id="a0783-104">'member' 조건부 멤버는 'Type Name' 형식으로 'base class member' 인터페이스 멤버를 구현할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="a0783-104">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
<span data-ttu-id="a0783-105">[조건부 특성](../language-reference/attributes/general.md#conditional-attribute) 은 인터페이스 구현에 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="a0783-105">The [Conditional attribute](../language-reference/attributes/general.md#conditional-attribute) cannot be used on the implementation of an interface.</span></span>  
  
<span data-ttu-id="a0783-106">다음 샘플에서는 CS0629를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a0783-106">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
