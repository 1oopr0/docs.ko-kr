# <a name="compiler-error-cs0112"></a><span data-ttu-id="3a898-103">컴파일러 오류 CS0112</span><span class="sxs-lookup"><span data-stu-id="3a898-103">Compiler Error CS0112</span></span>

<span data-ttu-id="3a898-104">'function' 정적 멤버는 override, virtual 또는 abstract로 표시할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="3a898-104">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="3a898-105">`override`, **virtual**또는 **abstract** 키워드를 사용하는 메서드 선언에서 **static** 키워드도 사용할 수는 없습니다.</span><span class="sxs-lookup"><span data-stu-id="3a898-105">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="3a898-106">자세한 내용은 [메서드](../programming-guide/classes-and-structs/methods.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="3a898-106">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="3a898-107">다음 샘플에서는 CS0112를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="3a898-107">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
