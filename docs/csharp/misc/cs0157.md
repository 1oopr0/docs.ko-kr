<span data-ttu-id="3978c-104">제어가 finally 절의 본문을 벗어날 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="3978c-104">Control cannot leave the body of a finally clause</span></span>  
  
 <span data-ttu-id="3978c-105">[finally](../language-reference/keywords/try-catch-finally.md) 절의 모든 문을 실행해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="3978c-105">All of the statements in a [finally](../language-reference/keywords/try-catch-finally.md) clause must execute.</span></span> <span data-ttu-id="3978c-106">자세한 내용은 [예외 및 예외 처리](../programming-guide/exceptions/index.md)를 참조 하세요.</span><span class="sxs-lookup"><span data-stu-id="3978c-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>

```csharp  
// CS0157.cs

namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
