# <a name="compiler-error-cs0271"></a><span data-ttu-id="2b130-103">컴파일러 오류 CS0271</span><span class="sxs-lookup"><span data-stu-id="2b130-103">Compiler Error CS0271</span></span>
<span data-ttu-id="2b130-104">get 접근자에 액세스할 수 없으므로 'property/indexer' 속성 또는 인덱서는 이 컨텍스트에서 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="2b130-104">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="2b130-105">이 오류는 액세스할 수 없는 `get` 접근자에 액세스하려고 할 때 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="2b130-105">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="2b130-106">이 오류를 해결하려면 접근자의 접근성을 향상하거나 호출 위치를 변경합니다.</span><span class="sxs-lookup"><span data-stu-id="2b130-106">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="2b130-107">자세한 내용은 [접근자 접근성](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) 및 [속성](../programming-guide/classes-and-structs/properties.md)을 참조 하세요.</span><span class="sxs-lookup"><span data-stu-id="2b130-107">For more information, see [Accessor Accessibility](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="2b130-108">다음 예제에서는 CS0271을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="2b130-108">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
