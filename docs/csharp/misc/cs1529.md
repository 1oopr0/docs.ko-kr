<span data-ttu-id="7e306-104">extern 별칭 선언을 제외하고 using 절은 네임스페이스에 정의된 다른 모든 요소보다 앞에 와야 합니다.</span><span class="sxs-lookup"><span data-stu-id="7e306-104">A using clause must precede all other elements defined in the namespace except extern alias declarations</span></span>

## <a name="example"></a><span data-ttu-id="7e306-106">예제</span><span class="sxs-lookup"><span data-stu-id="7e306-106">Example</span></span>  

 <span data-ttu-id="7e306-107">다음 샘플에서는 CS1529를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="7e306-107">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs  
namespace X  
{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}  
  
using System;                 // CS1529, place at the beginning of the file  
```
