# <a name="compiler-error-cs0167"></a>컴파일러 오류 CS0167
'delegate' 대리자에 Invoke 메서드가 없습니다.  
  
 다른 컴파일러로 만든 관리 되는 프로그램 (.NET 런타임을 사용 하는 프로그램)을 가져와서 사용 했습니다. 해당 컴파일러에서 잘못된 형식의 [대리자](../language-reference/builtin-types/reference-types.md)를 허용했습니다. 따라서 `Invoke` 메서드를 사용할 수 없습니다. 자세한 내용은 [대리자](../programming-guide/delegates/index.md)를 참조하세요.
