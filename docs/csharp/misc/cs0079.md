# <a name="compiler-error-cs0079"></a><span data-ttu-id="0ccfd-103">컴파일러 오류 CS0079</span><span class="sxs-lookup"><span data-stu-id="0ccfd-103">Compiler Error CS0079</span></span>
<span data-ttu-id="0ccfd-104">'event' 이벤트는 += 또는 -=의 왼쪽에만 올 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="0ccfd-104">The event 'event' can only appear on the left hand side of += or -=</span></span>  
  
 <span data-ttu-id="0ccfd-105">[이벤트](../language-reference/keywords/event.md) 를 잘못 호출했습니다.</span><span class="sxs-lookup"><span data-stu-id="0ccfd-105">An [event](../language-reference/keywords/event.md) was called incorrectly.</span></span> <span data-ttu-id="0ccfd-106">자세한 내용은 [이벤트](../programming-guide/events/index.md) 및 [대리자](../programming-guide/delegates/index.md)를 참조 하세요.</span><span class="sxs-lookup"><span data-stu-id="0ccfd-106">For more information, see [Events](../programming-guide/events/index.md) and [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="0ccfd-107">다음 샘플에서는 CS0079를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="0ccfd-107">The following sample generates CS0079:</span></span>  
  
```csharp  
// CS0079.cs  
using System;  
  
public delegate void MyEventHandler();  
  
public class Class1  
{  
   private MyEventHandler _e;  
  
   public event MyEventHandler Pow  
   {  
      add  
      {  
         _e += value;  
         Console.WriteLine("in add accessor");  
      }  
      remove  
      {  
         _e -= value;  
         Console.WriteLine("in remove accessor");  
      }  
   }  
  
   public void Handler()  
   {  
   }  
  
   public void Fire()  
   {  
      if (_e != null)  
      {  
         Pow();   // CS0079  
         // try the following line instead  
         // _e();  
      }  
   }  
  
   public static void Main()  
   {  
      Class1 p = new Class1();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p._e -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
      p.Pow -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
   }  
}  
```
