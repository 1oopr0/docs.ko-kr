 <span data-ttu-id="b5912-105">컴파일러가 [case](../language-reference/keywords/switch.md) 또는 **default** 문 없이 **switch** 블록을 검색했습니다.</span><span class="sxs-lookup"><span data-stu-id="b5912-105">The compiler detected a [switch](../language-reference/keywords/switch.md) block with no **case** or **default** statement.</span></span> <span data-ttu-id="b5912-106">`switch` 블록에는 하나 이상의 **case** 또는 **default** 문이 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="b5912-106">A `switch` block must have one or more **case** or **default** statements.</span></span>

```csharp  
{  
   public static void Main()  
   {  
      int i = 6;  
  
      switch(i)   // CS1522  
      {  
         // add something to the switch block, for example:  
         /*  
         case (5):  
            Console.WriteLine("5");  
            return;  
         default:  
            Console.WriteLine("not 5");  
            return;  
         */  
      }  
   }  
}  
```
