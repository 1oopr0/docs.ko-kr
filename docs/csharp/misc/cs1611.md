 <span data-ttu-id="df486-106">다음 샘플에서는 CS1611을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="df486-106">The following sample generates CS1611:</span></span>  
  
```csharp

      Test(1);  
   }  
}  
```
