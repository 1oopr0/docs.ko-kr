 <span data-ttu-id="4330f-105">[In](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) 또는 [out](../language-reference/keywords/out-parameter-modifier.md) 키워드는 [params](../language-reference/keywords/params.md) 키워드와 함께 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="4330f-105">The keywords [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) cannot be used with the [params](../language-reference/keywords/params.md) keyword.</span></span>  
  
 <span data-ttu-id="4330f-106">다음 샘플에서는 CS1611을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="4330f-106">The following sample generates CS1611:</span></span>  
  
```csharp

   public static void Test(params ref int[] a)   // CS1611, remove ref

      Test(1);  
   }  
}  
```
