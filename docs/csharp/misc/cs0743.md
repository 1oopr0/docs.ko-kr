# <a name="compiler-error-cs0743"></a><span data-ttu-id="e0ac7-103">컴파일러 오류 CS0743</span><span class="sxs-lookup"><span data-stu-id="e0ac7-103">Compiler Error CS0743</span></span>
<span data-ttu-id="e0ac7-104">키워드 'on'이 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="e0ac7-104">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="e0ac7-105">다음 예제와 같이 `join` 절에 대한 패턴은 `join`...`in`...`on`...`equals`입니다.</span><span class="sxs-lookup"><span data-stu-id="e0ac7-105">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```

## <a name="example"></a><span data-ttu-id="e0ac7-108">예제</span><span class="sxs-lookup"><span data-stu-id="e0ac7-108">Example</span></span>  
 <span data-ttu-id="e0ac7-109">다음 코드에서는 CS0743을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="e0ac7-109">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
