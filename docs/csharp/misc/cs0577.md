 <span data-ttu-id="fdb06-106">예를 들어 명시적 인터페이스 정의에 일부 특성을 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="fdb06-106">For example, you cannot use some attributes on an explicit interface definition.</span></span> <span data-ttu-id="fdb06-107">다음 샘플에서는 CS0577을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="fdb06-107">The following sample generates CS0577:</span></span>  
  
```csharp  
// CS0577.cs

{  
   void m();  
}  
  
public class MyClass : I  
{  
   [System.Diagnostics.Conditional("a")]   // CS0577  
   void I.m() {}  
}  
```
