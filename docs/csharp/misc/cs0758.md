<span data-ttu-id="79c02-104">두 부분 메서드(Partial Method) 선언 모두 params 매개 변수를 사용하거나 params 매개 변수를 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="79c02-104">Both partial method declarations must use a params parameter or neither may use a params parameter</span></span>
