## <a name="example"></a><span data-ttu-id="abfd4-109">예제</span><span class="sxs-lookup"><span data-stu-id="abfd4-109">Example</span></span>  

 <span data-ttu-id="abfd4-110">다음 예제에서는 CS1102를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="abfd4-110">The following example generates CS1102:</span></span>  
  
```csharp  
// cs1102.cs  
// Compile with: /target:library.  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(this out int i) {} // CS1102  
  
    //Single type parameter  
        public static void Test<T>(this out T t) {}// CS1102  
  
    //Multiple type parameters  
        public static void Test<T,U,V>(this out U u) {}// CS1102  
}  
```
