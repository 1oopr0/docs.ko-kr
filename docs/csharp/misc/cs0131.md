## <a name="example"></a><span data-ttu-id="a4420-109">예제</span><span class="sxs-lookup"><span data-stu-id="a4420-109">Example</span></span>  

 <span data-ttu-id="a4420-110">다음 샘플에서는 CS0131을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a4420-110">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="a4420-111">예제</span><span class="sxs-lookup"><span data-stu-id="a4420-111">Example</span></span>

{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;

    }
