## <a name="example"></a><span data-ttu-id="e4497-109">예제</span><span class="sxs-lookup"><span data-stu-id="e4497-109">Example</span></span>  
 <span data-ttu-id="e4497-110">다음 샘플에서는 CS0403을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="e4497-110">The following sample generates CS0403.</span></span>  
  
```csharp  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
