# <a name="compiler-error-cs1594"></a><span data-ttu-id="54887-103">컴파일러 오류 CS1594</span><span class="sxs-lookup"><span data-stu-id="54887-103">Compiler Error CS1594</span></span>
<span data-ttu-id="54887-104">'delegate' 대리자에 잘못된 인수가 있습니다.</span><span class="sxs-lookup"><span data-stu-id="54887-104">Delegate 'delegate' has some invalid arguments</span></span>  
  
 <span data-ttu-id="54887-105">[delegate](../language-reference/builtin-types/reference-types.md) 호출에 전달된 인수 형식이 대리자 선언의 매개 변수 형식에 맞지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="54887-105">The type of an argument passed to a [delegate](../language-reference/builtin-types/reference-types.md) invocation does not agree with the type of the parameter in the delegate declaration.</span></span>  
  
 <span data-ttu-id="54887-106">다음 샘플에서는 CS1594를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="54887-106">The following sample generates CS1594:</span></span>  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
