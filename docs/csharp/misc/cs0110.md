# <a name="compiler-error-cs0110"></a><span data-ttu-id="d1b57-103">컴파일러 오류 CS0110</span><span class="sxs-lookup"><span data-stu-id="d1b57-103">Compiler Error CS0110</span></span>

<span data-ttu-id="d1b57-104">'const declaration'에 대한 상수 값 계산에 순환 정의가 포함되어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="d1b57-104">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="d1b57-105">[const](../language-reference/keywords/const.md) 변수(`a`) 선언에서 역시 (`b`)를 참조하는 다른 const 변수(`a`)를 참조할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="d1b57-105">The declaration of a [const](../language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="d1b57-106">다음 샘플에서는 CS0110을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d1b57-106">The following sample generates CS0110:</span></span>  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
