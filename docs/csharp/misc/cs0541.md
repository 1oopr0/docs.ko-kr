# <a name="compiler-error-cs0541"></a><span data-ttu-id="1de5b-103">컴파일러 오류 CS0541</span><span class="sxs-lookup"><span data-stu-id="1de5b-103">Compiler Error CS0541</span></span>
<span data-ttu-id="1de5b-104">'declaration': 명시적 인터페이스 선언은 클래스나 구조체에서만 선언할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="1de5b-104">'declaration' : explicit interface declaration can only be declared in a class or struct</span></span>  
  
 <span data-ttu-id="1de5b-105">명시적 [인터페이스](../language-reference/keywords/interface.md) 선언을 [클래스](../language-reference/keywords/class.md) 또는 [구조체](../language-reference/builtin-types/struct.md)외부에서 찾았습니다.</span><span class="sxs-lookup"><span data-stu-id="1de5b-105">An explicit [interface](../language-reference/keywords/interface.md) declaration was found outside a [class](../language-reference/keywords/class.md) or [struct](../language-reference/builtin-types/struct.md).</span></span>  
  
 <span data-ttu-id="1de5b-106">다음 샘플에서는 CS0541을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="1de5b-106">The following sample generates CS0541:</span></span>  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }  
  
   interface IFace2 : IFace  
   {  
      void IFace.F();   // CS0541  
   }  
}  
```
