# <a name="compiler-error-cs1686"></a><span data-ttu-id="9c6f9-102">컴파일러 오류 CS1686</span><span class="sxs-lookup"><span data-stu-id="9c6f9-102">Compiler Error CS1686</span></span>

<span data-ttu-id="9c6f9-103">지역 'variable' 또는 해당 멤버의 주소를 가져와 무명 메서드 또는 람다 식 안에 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="9c6f9-103">Local 'variable' or its members cannot have their address taken and be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="9c6f9-104">이 오류는 변수를 사용하고 해당 주소를 사용하려고 하며 무명 메서드 내에서 이러한 작업 중 하나를 수행하는 경우에 생성됩니다.</span><span class="sxs-lookup"><span data-stu-id="9c6f9-104">This error is generated when you use a variable, and attempt to take its address, and one of these actions is done inside an anonymous method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9c6f9-105">예제</span><span class="sxs-lookup"><span data-stu-id="9c6f9-105">Example</span></span>  

 <span data-ttu-id="9c6f9-106">다음 샘플에서는 CS1686을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="9c6f9-106">The following sample generates CS1686.</span></span>  
  
```csharp  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
