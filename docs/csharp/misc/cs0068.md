<span data-ttu-id="f1e4f-104">'event': 인터페이스의 이벤트에는 이니셜라이저를 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="f1e4f-104">'event': event in interface cannot have initializer</span></span>  
  
 <span data-ttu-id="f1e4f-105">인터페이스의 이벤트에는 이니셜라이저를 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="f1e4f-105">An event in an interface cannot have an initializer.</span></span> <span data-ttu-id="f1e4f-106">자세한 내용은 [인터페이스](../programming-guide/interfaces/index.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="f1e4f-106">For more information, see [Interfaces](../programming-guide/interfaces/index.md).</span></span>  
  
 <span data-ttu-id="f1e4f-107">다음 샘플에서는 CS0068을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="f1e4f-107">The following sample generates CS0068:</span></span>  
  
```csharp  
// CS0068.cs

interface I  
{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068  
   // try the following line instead  
   // event MyDelegate d2;  
}  
  
class M  
{  
   event MyDelegate d = new MyDelegate(M.f);  
  
   public static void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
