# <a name="compiler-error-cs1918"></a><span data-ttu-id="e3312-102">컴파일러 오류 CS1918</span><span class="sxs-lookup"><span data-stu-id="e3312-102">Compiler Error CS1918</span></span>

<span data-ttu-id="e3312-103">형식이 'type'인 'name' 속성의 멤버는 값 형식이므로 개체 이니셜라이저를 사용하여 할당할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="e3312-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="e3312-104">이 오류는 개체 이니셜라이저를 사용하여 그 자체가 초기화되는 클래스의 속성인 구조체 형식의 속성을 초기화하려는 경우에 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="e3312-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e3312-105">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="e3312-105">To correct this error</span></span>  
  
1. <span data-ttu-id="e3312-106">개체 이니셜라이저에서 속성의 필드를 완전히 초기화해야 하는 경우 구조체를 클래스 형식으로 변경합니다.</span><span class="sxs-lookup"><span data-stu-id="e3312-106">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="e3312-107">그렇지 않으면 개체 이니셜라이저를 사용하여 개체를 만든 후 별도 메서드 호출에서 구조체 멤버를 초기화합니다.</span><span class="sxs-lookup"><span data-stu-id="e3312-107">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e3312-108">예제</span><span class="sxs-lookup"><span data-stu-id="e3312-108">Example</span></span>  

 <span data-ttu-id="e3312-109">다음 예제에서는 CS1918을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="e3312-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
