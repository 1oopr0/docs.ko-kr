<span data-ttu-id="01cdf-105">정수 리터럴이 나타내는 값 <xref:System.UInt64.MaxValue?displayProperty=nameWithType> 이 보다 큰 경우</span><span class="sxs-lookup"><span data-stu-id="01cdf-105">A value represented by an integer literal is greater than <xref:System.UInt64.MaxValue?displayProperty=nameWithType>.</span></span>
