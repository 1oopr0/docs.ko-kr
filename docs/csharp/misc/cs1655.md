# <a name="compiler-error-cs1655"></a><span data-ttu-id="47e09-103">컴파일러 오류 CS1655</span><span class="sxs-lookup"><span data-stu-id="47e09-103">Compiler Error CS1655</span></span>

<span data-ttu-id="47e09-104">'variable'은 'readonly variable type'이므로 해당 필드를 ref 또는 out 인수로 전달할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="47e09-104">Cannot pass fields of 'variable' as a ref or out argument because it is a 'readonly variable type'</span></span>  
  
 <span data-ttu-id="47e09-105">이 오류는 [foreach](../language-reference/keywords/foreach-in.md) 변수, [using](../language-reference/keywords/using-statement.md) 변수 또는 [fixed](../language-reference/keywords/fixed-statement.md) 변수의 멤버를 ref 또는 out 인수로 함수에 전달하려는 경우에 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="47e09-105">This error occurs if you are attempting to pass a member of a [foreach](../language-reference/keywords/foreach-in.md) variable, a [using](../language-reference/keywords/using-statement.md) variable, or a [fixed](../language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="47e09-106">해당 변수는 이러한 컨텍스트에서 읽기 전용으로 간주되므로 이 작업을 수행할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="47e09-106">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>  
  
 <span data-ttu-id="47e09-107">다음 샘플에서는 CS1655를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="47e09-107">The following sample generates CS1655:</span></span>  
  
```csharp  
// CS1655.cs  
struct S
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }  
  
  public static void Main()  
  {  
     S[] sa = new S[10];  
     foreach(S s in sa)  
     {  
        CMain.f(ref s.i);  // CS1655  
     }  
  }  
}  
```
