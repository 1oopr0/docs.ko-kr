# <a name="compiler-error-cs1641"></a><span data-ttu-id="b483b-103">컴파일러 오류 CS1641</span><span class="sxs-lookup"><span data-stu-id="b483b-103">Compiler Error CS1641</span></span>

<span data-ttu-id="b483b-104">고정 크기 버퍼 필드에는 필드 이름 뒤에 배열 크기 지정자를 사용해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="b483b-104">A fixed size buffer field must have the array size specifier after the field name</span></span>  
  
 <span data-ttu-id="b483b-105">정규 배열과 달리 고정 크기 버퍼에서는 선언 지점에서 상수 크기를 지정해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="b483b-105">Unlike regular arrays, fixed size buffers require a constant size to be specified at the declaration point.</span></span> <span data-ttu-id="b483b-106">이 오류를 해결하려면 양의 정수 리터럴 또는 상수 양의 정수를 추가하고 식별자 뒤에 대괄호를 넣습니다.</span><span class="sxs-lookup"><span data-stu-id="b483b-106">To resolve this error, add a positive integer literal or a constant positive integer and put the square brackets after the identifier.</span></span>  
  
 <span data-ttu-id="b483b-107">다음 샘플에서는 CS1641을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="b483b-107">The following sample generates CS1641:</span></span>  
  
```csharp  
// CS1641.cs  
// compile with: /unsafe /target:library  
unsafe struct S {  
   fixed int [] a;  // CS1641  
  
   // OK  
   fixed int b [10];  
   const int c = 10;  
   fixed int d [c];  
}  
```
