# <a name="compiler-error-cs0462"></a><span data-ttu-id="d50e7-103">컴파일러 오류 CS0462</span><span class="sxs-lookup"><span data-stu-id="d50e7-103">Compiler Error CS0462</span></span>

<span data-ttu-id="d50e7-104">상속된 멤버 'member1'과(와) 'member2'은(는) 'type' 형식에 동일한 서명이 있으므로 재정의할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="d50e7-104">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="d50e7-105">이 오류는 제네릭이 도입되어 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="d50e7-105">This error arises with the introduction of generics.</span></span> <span data-ttu-id="d50e7-106">일반적으로 한 클래스에서 한 메서드의 두 가지 버전을 동일한 서명으로 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="d50e7-106">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="d50e7-107">그러나 제네릭을 사용하면 특정 형식으로 인스턴스화된 경우 다른 메서드를 복제할 수 있는 제네릭 메서드를 지정할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="d50e7-107">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d50e7-108">예제</span><span class="sxs-lookup"><span data-stu-id="d50e7-108">Example</span></span>  

 <span data-ttu-id="d50e7-109">`C<int>` 가 인스턴스화된 경우 `F` 메서드의 두 가지 버전이 동일한 서명으로 만들어지므로 `D` 클래스의 재정의에서 재정의를 적용할 대상을 결정할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="d50e7-109">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="d50e7-110">다음 샘플에서는 CS0462를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d50e7-110">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>
{  
   public override void F(int t) {}   // CS0462  
}  
```
