# <a name="compiler-error-cs0040"></a>컴파일러 오류 CS0040

디버그 정보 파일을 만드는 동안 예기치 않은 오류가 발생했습니다. 'reason'  
  
 [-Debug](../language-reference/compiler-options/debug-compiler-option.md) 컴파일러 옵션을 사용 하 고 컴파일러가 .pdb 파일에 쓸 수 없음을 나타내는 경우이 오류가 발생할 수 있습니다. 이 오류를 해결하는 방법에는 Visual Studio 다시 설치, 컴파일러에 파일 또는 디렉터리에 대한 쓰기 권한이 있는지 확인, /debug로 컴파일하지 않음 등이 포함됩니다.
