 <span data-ttu-id="c214f-105">태그 내에서와 같이 cref 태그에 전달 된 문자열이 \<exception> 현재 빌드 환경에서 사용할 수 없는 멤버를 참조 합니다.</span><span class="sxs-lookup"><span data-stu-id="c214f-105">A string passed to a cref tag, for example, within an \<exception> tag, referred to a member that is not available within the current build environment.</span></span> <span data-ttu-id="c214f-106">cref 태그에 전달하는 문자열은 구문상 멤버 또는 필드의 올바른 이름이어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="c214f-106">The string that you pass to a cref tag must be the syntactically correct name of a member or field.</span></span>

```csharp  
{  
}

{  
   public static void Main()  
   {  
      try  
      {  
      }  
      catch(EClass)  
      {  
      }  
   }  
}  
```
