 <span data-ttu-id="ee5af-105">ComImport 특성을 지정하는 것은 COM 모듈에서 클래스에 대한 구현을 가져올 것을 의미합니다.</span><span class="sxs-lookup"><span data-stu-id="ee5af-105">Specifying the ComImport attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="ee5af-106">추가 메서드가 정의되지 않을 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="ee5af-106">Additional methods may not be defined.</span></span>

    ImageProperties i = new ImageProperties();
