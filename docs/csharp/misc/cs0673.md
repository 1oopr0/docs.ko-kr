 <span data-ttu-id="a7315-106">다음 샘플에서는 CS0673을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a7315-106">The following sample generates CS0673:</span></span>  
  
```csharp  
// CS0673.cs  
class MyClass  
{  
   public static void Main()  
   {  
      System.Type t = typeof(System.Void);   // CS0673  
      // try the following line instead  
      // System.Type t = typeof(void);  
   }  
}  
```
