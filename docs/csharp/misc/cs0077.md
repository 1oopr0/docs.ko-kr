# <a name="compiler-error-cs0077"></a><span data-ttu-id="22bce-103">컴파일러 오류 CS0077</span><span class="sxs-lookup"><span data-stu-id="22bce-103">Compiler Error CS0077</span></span>
<span data-ttu-id="22bce-104">as 연산자는 참조 형식 또는 nullable 형식과 함께 사용해야 합니다. 'int'는 null을 허용하지 않는 값 형식입니다.</span><span class="sxs-lookup"><span data-stu-id="22bce-104">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="22bce-105">[as](../language-reference/operators/type-testing-and-cast.md#as-operator) 연산자가 [값 형식](../language-reference/builtin-types/value-types.md)으로 전달되었습니다.</span><span class="sxs-lookup"><span data-stu-id="22bce-105">The [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../language-reference/builtin-types/value-types.md).</span></span> <span data-ttu-id="22bce-106">`as`는 [null](../language-reference/keywords/null.md)을 반환할 수 있기 때문에 [참조 형식](../language-reference/keywords/reference-types.md) 또는 [nullable 값 형식만](../language-reference/builtin-types/nullable-value-types.md)전달할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="22bce-106">Because `as` can return [null](../language-reference/keywords/null.md), it can only be passed a [reference type](../language-reference/keywords/reference-types.md) or a [nullable value type](../language-reference/builtin-types/nullable-value-types.md).</span></span>
  
 <span data-ttu-id="22bce-107">다음 샘플에서는 CS0077을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="22bce-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
