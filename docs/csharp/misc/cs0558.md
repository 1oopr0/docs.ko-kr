# <a name="compiler-error-cs0558"></a>컴파일러 오류 CS0558
'operator' 사용자 정의 연산자는 static 및 public으로 선언해야 합니다.  
  
 **static** 및 **public** 액세스 [한정자](../language-reference/keywords/index.md) 는 둘 다 사용자 정의 연산자에 지정되어야 합니다.  
  
 다음 샘플에서는 CS0558을 생성합니다.  
  
```csharp  
// CS0558.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
