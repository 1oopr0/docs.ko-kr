# <a name="compiler-error-cs1662"></a><span data-ttu-id="637b1-103">컴파일러 오류 CS1662</span><span class="sxs-lookup"><span data-stu-id="637b1-103">Compiler Error CS1662</span></span>

<span data-ttu-id="637b1-104">블록의 반환 형식 중 일부를 암시적으로 대리자 반환 형식으로 변환할 수 없으므로 무명 메서드 블록을 'delegate type' 대리자 형식으로 변환할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="637b1-104">Cannot convert anonymous method block to delegate type 'delegate type' because some of the return types in the block are not implicitly convertible to the delegate return type</span></span>

<span data-ttu-id="637b1-105">이 오류는 무명 메서드 블록의 return 문에 암시적으로 대리자의 반환 형식으로 변환할 수 없는 형식이 있는 경우 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="637b1-105">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>

<span data-ttu-id="637b1-106">다음 샘플에서는 CS1662를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="637b1-106">The following sample generates CS1662:</span></span>

```csharp
// CS1662.cs

delegate int MyDelegate(int i);

class C
{

  public static void Main()
  {
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662
     // Try this instead:
     // MyDelegate d = delegate(int i) { return (int)1.0; };
  }
}
```
