 <span data-ttu-id="aaabd-105">**이벤트** 정의에서 **add** 또는 [remove](../language-reference/keywords/event.md) 키워드는 본문이 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="aaabd-105">An **add** or **remove** keyword in an [event](../language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="aaabd-106">자세한 내용은 [이벤트](../programming-guide/events/index.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="aaabd-106">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="aaabd-107">다음 샘플에서는 CS0073을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="aaabd-107">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
