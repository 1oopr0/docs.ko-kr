 <span data-ttu-id="490ba-105">[인터페이스](../language-reference/keywords/interface.md)에서 이벤트의 접근자 함수를 정의할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="490ba-105">You cannot define an event's accessor functions in an [interface](../language-reference/keywords/interface.md).</span></span> <span data-ttu-id="490ba-106">자세한 내용은 [이벤트](../programming-guide/events/index.md) 및 [인터페이스](../programming-guide/interfaces/index.md)를 참조 하세요.</span><span class="sxs-lookup"><span data-stu-id="490ba-106">For more information, see [Events](../programming-guide/events/index.md) and [Interfaces](../programming-guide/interfaces/index.md).</span></span>  
  
 <span data-ttu-id="490ba-107">다음 샘플에서는 CS0069를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="490ba-107">The following sample generates CS0069:</span></span>

```csharp  
   event EventHandler Click { remove {} }   // CS0069  
   event EventHandler Click2;   // OK  
}  
```
