# <a name="compiler-error-cs1528"></a><span data-ttu-id="b17d3-103">컴파일러 오류 CS1528</span><span class="sxs-lookup"><span data-stu-id="b17d3-103">Compiler Error CS1528</span></span>
<span data-ttu-id="b17d3-104">; 또는 =가 필요합니다. 선언에서는 생성자 인수를 지정할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="b17d3-104">Expected ; or = (cannot specify constructor arguments in declaration)</span></span>  
  
 <span data-ttu-id="b17d3-105">클래스에 대한 참조가 클래스에 대한 개체가 생성되는 것처럼 형성되었습니다.</span><span class="sxs-lookup"><span data-stu-id="b17d3-105">A reference to a class was formed as if an object to the class was being created.</span></span> <span data-ttu-id="b17d3-106">예를 들어 생성자에 변수를 전달하려는 시도가 있었습니다.</span><span class="sxs-lookup"><span data-stu-id="b17d3-106">For example, there was an attempt to pass a variable to a constructor.</span></span> <span data-ttu-id="b17d3-107">클래스의 개체를 만들려면 [new](../language-reference/operators/new-operator.md) 연산자를 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="b17d3-107">Use the [new](../language-reference/operators/new-operator.md) operator to create an object of a class.</span></span>  
  
 <span data-ttu-id="b17d3-108">다음 샘플에서는 CS1528을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="b17d3-108">The following sample generates CS1528:</span></span>  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```
