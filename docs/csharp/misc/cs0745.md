 <span data-ttu-id="7ea05-105">`group` 절에 대한 패턴은 다음 예제와 같이 `group...by` 뒤에 선택적 `into`가 오는 것입니다.</span><span class="sxs-lookup"><span data-stu-id="7ea05-105">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>
