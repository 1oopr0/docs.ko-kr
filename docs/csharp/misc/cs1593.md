# <a name="compiler-error-cs1593"></a><span data-ttu-id="c9fe3-103">컴파일러 오류 CS1593</span><span class="sxs-lookup"><span data-stu-id="c9fe3-103">Compiler Error CS1593</span></span>
<span data-ttu-id="c9fe3-104">'del' 대리자는 인수를 'number'개 사용하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="c9fe3-104">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="c9fe3-105">[delegate](../language-reference/builtin-types/reference-types.md) 호출에 전달된 인수 개수가 대리자 선언의 매개 변수 개수에 맞지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="c9fe3-105">The number of arguments passed to a [delegate](../language-reference/builtin-types/reference-types.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="c9fe3-106">다음 샘플에서는 CS1593을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="c9fe3-106">The following sample generates CS1593:</span></span>  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
