# <a name="compiler-error-cs0584"></a>컴파일러 오류 CS0584

내부 컴파일러 오류: 'stage' 단계 'symbol' 기호
  
 컴파일러가 예기치 않은 구문을 분석할 수 없어 실패하는지 여부를 확인합니다. 그렇지 않은 경우 [저희에 게 문의](/visualstudio/ide/feedback-options)하세요.

> [!NOTE]
> 이 컴파일러 오류는 Roslyn에서 더 이상 사용되지 않습니다.
