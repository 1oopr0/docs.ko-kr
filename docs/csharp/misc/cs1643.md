<span data-ttu-id="46c02-104">코드 경로 중 일부에서만 'type!' 형식의 메서드에 있는 값을 반환합니다.</span><span class="sxs-lookup"><span data-stu-id="46c02-104">Not all code paths return a value in method of type 'type!'</span></span>  
  
 <span data-ttu-id="46c02-105">이 오류는 대리자 본문에 return 문이 없거나 컴파일러에서 확인할 수 없는 return 문이 있는 경우 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="46c02-105">This error occurs if a delegate body does not have a return statement, or has a return statement that the compiler is unable to verify will be reached.</span></span> <span data-ttu-id="46c02-106">아래 예제에서 컴파일러는 무명 메서드 블록이 항상 값을 반환하는지 확인하기 위해 분기 조건의 결과를 예측하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="46c02-106">In the example below, the compiler does not attempt to predict the result of the branching condition in order to verify that the anonymous method block always returns a value.</span></span>

 <span data-ttu-id="46c02-108">다음 샘플에서는 CS1643을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="46c02-108">The following sample generates CS1643:</span></span>

```csharp  
{  
    static void Main()  
    {  
        MyDelegate d = delegate  
        {                 // CS1643

            if (i == 0)  
                return 1;  
        };  
    }  
}  
```
