## <a name="example"></a><span data-ttu-id="89620-110">예제</span><span class="sxs-lookup"><span data-stu-id="89620-110">Example</span></span>

```csharp

namespace TestNamespace
{
    public delegate void Del();

    class Test
    {
        static int Main()
        {
            bool b1 = (() => { }) is Del;   // CS0837
            bool b2 = delegate() { } is Del;// CS0837

        }
    }
}
```
