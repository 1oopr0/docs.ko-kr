<span data-ttu-id="3de4f-104">null을 허용하지 않는 값 형식이므로 null을 'type'으로 변환할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="3de4f-104">Cannot convert null to 'type' because it is a non-nullable value type</span></span>  
  
 <span data-ttu-id="3de4f-105">컴파일러는 `null` 값 형식에 할당할 수 없습니다. `null` [참조 형식](../language-reference/keywords/reference-types.md) 또는 [nullable 값 형식](../language-reference/builtin-types/nullable-value-types.md)에만 할당할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="3de4f-105">The compiler cannot assign `null` to a value type; `null` can only be assigned to a [reference type](../language-reference/keywords/reference-types.md) or to a [nullable value type](../language-reference/builtin-types/nullable-value-types.md).</span></span> <span data-ttu-id="3de4f-106">[struct](../language-reference/builtin-types/struct.md) 가 값 형식입니다.</span><span class="sxs-lookup"><span data-stu-id="3de4f-106">[struct](../language-reference/builtin-types/struct.md) is a value type.</span></span>
  
 <span data-ttu-id="3de4f-107">다음 샘플에서는 CS0037을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="3de4f-107">The following sample generates CS0037:</span></span>
