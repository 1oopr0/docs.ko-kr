# <a name="compiler-error-cs0466"></a><span data-ttu-id="d55c8-103">컴파일러 오류 CS0466</span><span class="sxs-lookup"><span data-stu-id="d55c8-103">Compiler Error CS0466</span></span>

<span data-ttu-id="d55c8-104">'method1'이 없어 'method2'에 params 매개 변수를 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="d55c8-104">'method1' should not have a params parameter since 'method2' does not</span></span>  
  
 <span data-ttu-id="d55c8-105">구현된 인터페이스가 `params` 매개 변수를 사용할 수 없는 경우 클래스 맴버에서 해당 매개 변수를 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="d55c8-105">You cannot use `params` parameter on a class member if the implemented interface doesn't use it.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d55c8-106">예제</span><span class="sxs-lookup"><span data-stu-id="d55c8-106">Example</span></span>  

 <span data-ttu-id="d55c8-107">다음 샘플에서는 CS0466을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d55c8-107">The following sample generates CS0466.</span></span>  
  
```csharp  
// CS0466.cs  
interface I  
{  
   void F1(params int[] a);  
   void F2(int[] a);  
}  
  
class C : I  
{  
   void I.F1(params int[] a) {}  
   void I.F2(params int[] a) {}   // CS0466  
   void I.F2(int[] a) {}   // OK  
  
   public static void Main()  
   {  
      I i = (I) new C();  
  
      i.F1(new int[] {1, 2} );  
      i.F2(new int[] {1, 2} );  
   }  
}  
```
