# <a name="compiler-error-cs0214"></a><span data-ttu-id="694e7-103">컴파일러 오류 CS0214</span><span class="sxs-lookup"><span data-stu-id="694e7-103">Compiler Error CS0214</span></span>
<span data-ttu-id="694e7-104">포인터와 고정 크기 버퍼는 안전하지 않은 컨텍스트에서만 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="694e7-104">Pointers and fixed size buffers may only be used in an unsafe context</span></span>  
  
 <span data-ttu-id="694e7-105">포인터는 [안전하지 않은](../language-reference/keywords/unsafe.md) 키워드에만 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="694e7-105">Pointers can only be used with the [unsafe](../language-reference/keywords/unsafe.md) keyword.</span></span> <span data-ttu-id="694e7-106">자세한 내용은 [안전하지 않은 코드 및 포인터](../programming-guide/unsafe-code-pointers/index.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="694e7-106">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="694e7-107">다음 샘플에서는 CS0214를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="694e7-107">The following sample generates CS0214:</span></span>  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
