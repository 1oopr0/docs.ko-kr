<span data-ttu-id="098f7-104">++ 또는 -- 연산자의 매개 변수 형식은 포함하는 형식이어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="098f7-104">The parameter type for ++ or -- operator must be the containing type</span></span>  
  
 <span data-ttu-id="098f7-105">연산자 오버로드에 대한 메서드 선언은 다음의 특정 지침을 따라야 합니다.</span><span class="sxs-lookup"><span data-stu-id="098f7-105">The method declaration for an operator overload must follow certain guidelines.</span></span> <span data-ttu-id="098f7-106">++ 및 -- 연산자의 경우 매개 변수는 연산자가 오버로드되는 형식과 동일한 형식이어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="098f7-106">For the ++ and -- operators, it is required that the parameter be of the same type as the type in which the operator is being overloaded.</span></span>  
  
## <a name="example"></a><span data-ttu-id="098f7-107">예제</span><span class="sxs-lookup"><span data-stu-id="098f7-107">Example</span></span>  

 <span data-ttu-id="098f7-108">다음 샘플에서는 CS0559를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="098f7-108">The following sample generates CS0559:</span></span>  
  
```csharp  
// CS0559.cs  
// compile with: /target:library  
public class iii  
{  
   public static implicit operator int(iii x)  
   {  
      return 0;  
   }  
  
   public static implicit operator iii(int x)  
   {  
      return null;  
   }  
  
   public static int operator ++(int aa)   // CS0559  
   // try the following line instead  
   // public static iii operator ++(iii aa)  
   {  
      return (iii)0;  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="098f7-109">예제</span><span class="sxs-lookup"><span data-stu-id="098f7-109">Example</span></span>  

 <span data-ttu-id="098f7-110">다음 샘플에서는 CS0559를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="098f7-110">The following sample generates CS0559.</span></span>  
  
```csharp  
// CS0559_b.cs  
// compile with: /target:library  
public struct S  
{  
   public static S operator ++(S? s) { return new S(); }   // CS0559  
   public static S operator --(S? s) { return new S(); }   // CS0559  
}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
}  
```
