# <a name="compiler-warning-level-3-cs0219"></a><span data-ttu-id="396ce-103">컴파일러 경고(수준 3) CS0219</span><span class="sxs-lookup"><span data-stu-id="396ce-103">Compiler Warning (level 3) CS0219</span></span>
<span data-ttu-id="396ce-104">'variable' 변수가 할당되었지만 변수 값이 사용되지 않았습니다.</span><span class="sxs-lookup"><span data-stu-id="396ce-104">The variable 'variable' is assigned but its value is never used</span></span>

 <span data-ttu-id="396ce-105">변수를 선언하고 할당하지만 사용하지 않을 경우 컴파일러에서 수준 3 경고를 실행합니다.</span><span class="sxs-lookup"><span data-stu-id="396ce-105">The compiler issues a level-three warning, when you declare and assign a variable, but do not use it.</span></span>

 > [!NOTE]
 > <span data-ttu-id="396ce-106">컴파일러는 변수 값이 컴파일 타임 상수인 경우에만이 경고를 생성 합니다.</span><span class="sxs-lookup"><span data-stu-id="396ce-106">The compiler generates this warning only when the variable value is a compile-time constant.</span></span> <span data-ttu-id="396ce-107">비상수 식 또는 메서드 결과를 지역 변수에 할당 하면 디버거에서 해당 식을 더 쉽게 관찰할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="396ce-107">Assigning a non-constant expression or method result to a local variable makes it easier to observe those expressions in the debugger.</span></span> <span data-ttu-id="396ce-108">또한 해당 변수에 연결할 수 있는 동안 가비지 수집을 방지 하는 결과에 연결할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="396ce-108">It also makes the result reachable, preventing garbage collection while that variable is reachable.</span></span>

 <span data-ttu-id="396ce-109">다음 샘플에서는 경고가 생성 되지 않는 경우와 시기를 보여 줍니다.</span><span class="sxs-lookup"><span data-stu-id="396ce-109">The following sample shows the cases when and when not the warning is generated:</span></span>

```csharp
// CS0219.cs
// compile with: /W:3
public class MyClass
{
   public static void Main()
   {
      var interpolated = "Interpolated";
      var a = 0; // CS0219
      int b = GetZero(); // Doesn't generate a warning.
      var c = "Regular string"; // CS0219
      var d = $"Constant interpolated string"; // Doesn't generate a warning.
      var e = $"{interpolated} string"; // Doesn't generate a warning.
   }

   private static int GetZero()
   {
       return 0;
   }
}  
```
