# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="fb4ec-103">컴파일러 경고 (수준 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="fb4ec-103">Compiler warning (level 1) CS0612</span></span>

<span data-ttu-id="fb4ec-104">'member'는 사용되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="fb4ec-104">'member' is obsolete</span></span>  
  
 <span data-ttu-id="fb4ec-105">클래스 디자이너에서 [사용 되지 않는 특성](../language-reference/attributes/general.md#obsolete-attribute)을 가진 멤버를 표시 했습니다.</span><span class="sxs-lookup"><span data-stu-id="fb4ec-105">The class designer marked a member with the [Obsolete attribute](../language-reference/attributes/general.md#obsolete-attribute).</span></span> <span data-ttu-id="fb4ec-106">이 경우 멤버가 이후 버전의 클래스에서 지원되지 않을 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="fb4ec-106">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="fb4ec-107">다음 샘플에서는 사용되지 않는 멤버에 액세스할 경우 CS0612가 발생하는 방식을 보여 줍니다.</span><span class="sxs-lookup"><span data-stu-id="fb4ec-107">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
