# <a name="compiler-error-cs0213"></a><span data-ttu-id="941ae-103">컴파일러 오류 CS0213</span><span class="sxs-lookup"><span data-stu-id="941ae-103">Compiler Error CS0213</span></span>

<span data-ttu-id="941ae-104">이미 고정된 식의 주소를 가져오는 데 fixed 문을 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="941ae-104">You cannot use the fixed statement to take the address of an already fixed expression</span></span>  
  
 <span data-ttu-id="941ae-105">[안전하지 않은](../language-reference/keywords/unsafe.md) 메서드 또는 매개 변수의 지역 변수가 이미 스택에 고정되어 있으므로 [고정된](../language-reference/keywords/fixed-statement.md) 식에서 이러한 두 변수 중 하나의 주소를 가져올 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="941ae-105">A local variable in an [unsafe](../language-reference/keywords/unsafe.md) method or a parameter is already fixed (on the stack), so you cannot take the address of either of these two variables in a [fixed](../language-reference/keywords/fixed-statement.md) expression.</span></span> <span data-ttu-id="941ae-106">자세한 내용은 [안전하지 않은 코드 및 포인터](../programming-guide/unsafe-code-pointers/index.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="941ae-106">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="941ae-107">예제</span><span class="sxs-lookup"><span data-stu-id="941ae-107">Example</span></span>  

 <span data-ttu-id="941ae-108">다음 샘플에서는 CS0213을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="941ae-108">The following sample generates CS0213.</span></span>  
  
```csharp  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```
