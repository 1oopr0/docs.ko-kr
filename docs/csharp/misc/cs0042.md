# <a name="compiler-error-cs0042"></a>컴파일러 오류 CS0042
디버그 정보 파일 'file'을 만드는 동안 예기치 않은 오류가 발생했습니다. 'reason'  
  
 컴파일러에서 디버그 정보를 만들 수 없습니다. *reason* 에 오류 조건에 대한 추가 정보가 있습니다.
