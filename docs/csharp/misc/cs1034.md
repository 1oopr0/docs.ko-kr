# <a name="compiler-error-cs1034"></a>컴파일러 오류 CS1034

컴파일러 한계를 초과했습니다. 줄은 'number'자를 초과할 수 없습니다.  
  
 한 줄에 허용되는 문자 수의 제한은 16,777,214자입니다.
