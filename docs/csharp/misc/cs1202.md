# <a name="compiler-warning-level-1-cs1202"></a>컴파일러 경고(수준 1) CS1202
'invalid feature' 기능은 사용되지 않습니다. 대신 'valid feature'를 사용하세요.  
  
 사용하려는 기능은 이제 더 이상 사용되지 않습니다. 대신 유효한 기능을 사용하도록 코드를 업데이트합니다.
