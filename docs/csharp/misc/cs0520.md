# <a name="compiler-error-cs0520"></a>컴파일러 오류 CS0520
미리 정의된 'name' 형식이 잘못 선언되었습니다.  
  
 컴파일러에서 필요한 하나 이상의 파일이 없습니다. .NET 런타임을 다시 설치 합니다.
