# <a name="compiler-error-cs0534"></a><span data-ttu-id="fcd24-103">컴파일러 오류 CS0534</span><span class="sxs-lookup"><span data-stu-id="fcd24-103">Compiler Error CS0534</span></span>

<span data-ttu-id="fcd24-104">'function1'은 상속된 추상 멤버 'function2'를 구현하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="fcd24-104">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="fcd24-105">클래스가 추상이 아닌 경우 클래스는 기본 클래스의 모든 [추상](../language-reference/keywords/abstract.md) 멤버를 구현해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="fcd24-105">A class is required to implement all the [abstract](../language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="fcd24-106">다음 샘플에서는 CS0534를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="fcd24-106">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
