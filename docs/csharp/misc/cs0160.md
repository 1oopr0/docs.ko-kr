# <a name="compiler-error-cs0160"></a><span data-ttu-id="6dcb7-103">컴파일러 오류 CS0160</span><span class="sxs-lookup"><span data-stu-id="6dcb7-103">Compiler Error CS0160</span></span>

<span data-ttu-id="6dcb7-104">이전의 catch 절에서 이 형식이나 상위 형식('type')의 예외를 모두 catch합니다.</span><span class="sxs-lookup"><span data-stu-id="6dcb7-104">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="6dcb7-105">일련의 [catch](../language-reference/keywords/try-catch.md) 문을 파생의 내림차순으로 정렬해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="6dcb7-105">A series of [catch](../language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="6dcb7-106">예를 들어 가장 많이 파생된 개체가 첫 번째로 나타나야 합니다.</span><span class="sxs-lookup"><span data-stu-id="6dcb7-106">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="6dcb7-107">자세한 내용은 [예외 및 예외 처리](../programming-guide/exceptions/index.md)를 참조 하세요.</span><span class="sxs-lookup"><span data-stu-id="6dcb7-107">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="6dcb7-108">다음 샘플에서는 CS0160을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="6dcb7-108">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
