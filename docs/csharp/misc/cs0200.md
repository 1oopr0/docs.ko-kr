<span data-ttu-id="d3428-105">[속성](../programming-guide/classes-and-structs/using-properties.md)에 값을 할당 하려고 했지만 속성에 set 접근자가 없거나 할당이 생성자 외부에 있는 경우</span><span class="sxs-lookup"><span data-stu-id="d3428-105">An attempt was made to assign a value to a [property](../programming-guide/classes-and-structs/using-properties.md), but the property does not have a set accessor or the assignment was outside of the constructor.</span></span> <span data-ttu-id="d3428-106">set 접근자를 추가하여 오류를 해결합니다.</span><span class="sxs-lookup"><span data-stu-id="d3428-106">Resolve the error by adding a set accessor.</span></span> <span data-ttu-id="d3428-107">자세한 내용은 [읽기/쓰기 속성을 선언 하 고 사용 하는 방법](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md)을 참조 하세요.</span><span class="sxs-lookup"><span data-stu-id="d3428-107">For more information, see [How to declare and use read-write properties](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).</span></span>

## <a name="examples"></a><span data-ttu-id="d3428-108">예제</span><span class="sxs-lookup"><span data-stu-id="d3428-108">Examples</span></span>

<span data-ttu-id="d3428-109">다음 샘플에서는 CS0200을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d3428-109">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="d3428-110">다음 샘플에서는 [자동으로 구현 된 속성](../programming-guide/classes-and-structs/auto-implemented-properties.md) 및 [개체 이니셜라이저](../programming-guide/classes-and-structs/object-and-collection-initializers.md) 를 사용 하 고 여전히 CS0200을 생성 합니다.</span><span class="sxs-lookup"><span data-stu-id="d3428-110">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md) and [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md) and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="d3428-111">읽기 전용인 속성 또는 인덱서에 ' property '를 할당 하려면 set 접근자를 추가 하거나 개체의 생성자에 값을 할당 합니다.</span><span class="sxs-lookup"><span data-stu-id="d3428-111">To assign to a property or indexer 'property' that's read-only, add a set accessor or assign the value in the object's constructor.</span></span>

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
