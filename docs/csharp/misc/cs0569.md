# <a name="compiler-error-cs0569"></a>컴파일러 오류 CS0569

'method2': 'method1'은 해당 언어에서 지원되지 않으므로 재정의할 수 없습니다.  
  
 컴파일러가 재정의하려는 메서드를 지원하지 않을 때와 다른 언어로 작성된 기본 클래스에서 파생될 때 이 오류가 발생합니다.
