## <a name="example"></a><span data-ttu-id="47aeb-108">예제</span><span class="sxs-lookup"><span data-stu-id="47aeb-108">Example</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="47aeb-110">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="47aeb-110">To correct this error</span></span>

- <span data-ttu-id="47aeb-112">클래스 `B`가 `A`로부터 상속하고 `MyClass`가 `B`로부터 상속하도록 합니다.</span><span class="sxs-lookup"><span data-stu-id="47aeb-112">Make class `B` inherit from `A`, and `MyClass` inherit from `B`:</span></span>

    ```csharp
    public class MyClass : A, B {}
    ```
