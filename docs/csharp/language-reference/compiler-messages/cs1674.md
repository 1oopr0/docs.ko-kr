# <a name="compiler-error-cs1674"></a><span data-ttu-id="31e64-103">컴파일러 오류 CS1674</span><span class="sxs-lookup"><span data-stu-id="31e64-103">Compiler Error CS1674</span></span>

<span data-ttu-id="31e64-104">‘T’: Using 문에 사용된 형식은 암시적으로 ‘System.IDisposable’로 변환 가능해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="31e64-104">'T': type used in a using statement must be implicitly convertible to 'System.IDisposable'</span></span>  
  
 <span data-ttu-id="31e64-105">[Using 문](../keywords/using-statement.md)은 `using` 블록의 끝에서 개체를 삭제하는 데 사용하기 위한 용도이므로 삭제 가능한 형식만 이러한 문에 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="31e64-105">The [using Statement](../keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="31e64-106">예를 들어, 값 형식은 삭제 가능하지 않으며 클래스로 제한되지 않는 형식 매개 변수는 삭제 가능한 것으로 간주되지 않을 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="31e64-106">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="31e64-107">예제</span><span class="sxs-lookup"><span data-stu-id="31e64-107">Example</span></span>  

 <span data-ttu-id="31e64-108">다음 샘플에서는 CS1674 오류가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="31e64-108">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674.cs  
class C  
{  
   public static void Main()  
   {  
      int a = 0;  
      a++;  
  
      using (a) {}   // CS1674  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="31e64-109">예제</span><span class="sxs-lookup"><span data-stu-id="31e64-109">Example</span></span>  

 <span data-ttu-id="31e64-110">다음 샘플에서는 CS1674 오류가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="31e64-110">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
