# <a name="compiler-error-cs8403"></a>컴파일러 오류 CS8403

‘{IAsyncEnumerable\<T>|IAsyncEnumerator\<T>}’를 반환하려면 반복기 블록을 포함하는 ‘method’ 메서드가 ‘async’여야 합니다.

## <a name="to-correct-this-error"></a>이 오류를 해결하려면

`async` 한정자를 사용하여 메서드를 표시합니다.
