 <span data-ttu-id="168e4-108">다음 샘플에서는 `MyClass`에 제공되는 접근성 한정자가 없어 접근성이 기본값인 `private`으로 설정되기 때문에 CS0050이 생성됩니다.</span><span class="sxs-lookup"><span data-stu-id="168e4-108">The following sample generates CS0050 because no accessibility modifier is supplied for `MyClass`, and its accessibility therefore defaults to `private`:</span></span>

        return new MyClass();
