# <a name="compiler-error-cs0006"></a>컴파일러 오류 CS0006

메타데이터 파일 ‘dll_name’을(를) 찾을 수 없습니다.
  
 프로그램이 컴파일되었으며 메타데이터를 포함하는 파일 이름이 명시적으로 전달되었지만 .dll을 찾을 수 없습니다. 자세한 내용은 [-reference(C# 컴파일러 옵션)](../compiler-options/reference-compiler-option.md)를 참조하세요.
