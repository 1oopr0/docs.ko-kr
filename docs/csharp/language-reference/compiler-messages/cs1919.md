<span data-ttu-id="d2553-104">안전하지 않은 형식 'type name'은 개체를 만드는 데 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="d2553-104">Unsafe type 'type name' cannot be used in object creation.</span></span>

## <a name="example"></a><span data-ttu-id="d2553-111">예제</span><span class="sxs-lookup"><span data-stu-id="d2553-111">Example</span></span>
