# <a name="compiler-error-cs1921"></a><span data-ttu-id="711cd-103">컴파일러 오류 CS1921</span><span class="sxs-lookup"><span data-stu-id="711cd-103">Compiler Error CS1921</span></span>

<span data-ttu-id="711cd-104">‘method’에 가장 일치하는 오버로드된 메서드에 이니셜라이저 요소의 잘못된 시그니처가 있습니다.</span><span class="sxs-lookup"><span data-stu-id="711cd-104">The best overloaded method match for 'method' has wrong signature for the initializer element.</span></span> <span data-ttu-id="711cd-105">초기화 가능한 Add는 액세스 가능한 인스턴스 메서드여야 합니다.</span><span class="sxs-lookup"><span data-stu-id="711cd-105">The initializable Add must be an accessible instance method.</span></span>  
  
 <span data-ttu-id="711cd-106">이 오류는 공용 비정적 `Add` 메서드가 없는 클래스와 함께 컬렉션 이니셜라이저를 사용하려고 할 때 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="711cd-106">This error is generated when you try to use a collection initializer with a class that has no public non-static `Add` method.</span></span> <span data-ttu-id="711cd-107">보호 수준(`private`, `protected`, `internal`) 때문에 `Add` 메서드에 액세스할 수 없는 경우에는 CS0122 오류가 발생합니다. 따라서 이 오류는 메서드가 `static`으로 정의되어 있음을 의미할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="711cd-107">If the `Add` method is not accessible because of its protection level (`private`, `protected`, `internal`) then you will get CS0122, so that this error probably means that the method is defined as `static`.</span></span>
