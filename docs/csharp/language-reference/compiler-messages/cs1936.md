<span data-ttu-id="5d3a8-106">소스 형식을 쿼리하려면 사용자가 쿼리에서 호출하는 표준 쿼리 연산자 메서드를 해당 형식에서 구현해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="5d3a8-106">In order to query a source type, that type must implement the standard query operator methods that you are invoking in the query.</span></span> <span data-ttu-id="5d3a8-107">적절한 `using` 지시문을 포함하는 범위에 표시되는 클래스 멤버 또는 확장 메서드 형식으로 구현할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="5d3a8-107">The implementation can be either in the form of class members or extension methods that are brought into scope with the appropriate `using` directive.</span></span>

## <a name="example"></a><span data-ttu-id="5d3a8-111">예제</span><span class="sxs-lookup"><span data-stu-id="5d3a8-111">Example</span></span>

```csharp

{  
    static int Main()

        object obj;
```

<span data-ttu-id="5d3a8-113">일반적으로 이 오류는 이러한 개체 컬렉션 대신 실수로 일부 형식의 개체를 쿼리하는 경우에 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="5d3a8-113">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
