# <a name="compiler-warning-level-2-cs0618"></a><span data-ttu-id="af8c0-103">컴파일러 경고(수준 2) CS0618</span><span class="sxs-lookup"><span data-stu-id="af8c0-103">Compiler Warning (level 2) CS0618</span></span>

<span data-ttu-id="af8c0-104">'member'는 사용되지 않습니다. 'text'</span><span class="sxs-lookup"><span data-stu-id="af8c0-104">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="af8c0-105">클래스 멤버가 `Obsolete` 특성으로 표시되어 클래스 멤버를 참조할 때 경고가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="af8c0-105">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="af8c0-106">자세한 내용은 [공통 특성](../attributes/global.md)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="af8c0-106">For more information, see [Common Attributes](../attributes/global.md).</span></span>  
  
 <span data-ttu-id="af8c0-107">다음 샘플에서는 CS0618을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="af8c0-107">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
