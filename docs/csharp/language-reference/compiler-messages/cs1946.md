# <a name="compiler-error-cs1946"></a><span data-ttu-id="3c659-102">컴파일러 오류 CS1946</span><span class="sxs-lookup"><span data-stu-id="3c659-102">Compiler Error CS1946</span></span>

<span data-ttu-id="3c659-103">무명 메서드 식을 식 트리로 변환할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="3c659-103">An anonymous method expression cannot be converted to an expression tree.</span></span>

<span data-ttu-id="3c659-104">[무명 메서드](../operators/delegate-operator.md)는 문 집합을 나타내지만 식 트리에 문이 있으면 안 됩니다.</span><span class="sxs-lookup"><span data-stu-id="3c659-104">An [anonymous method](../operators/delegate-operator.md) represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="3c659-105">따라서 식 트리에서 무명 메서드를 나타낼 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="3c659-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

<span data-ttu-id="3c659-106">이 오류를 해결하려면 무명 메서드를 [람다 식](../operators/lambda-expressions.md)으로 변경합니다.</span><span class="sxs-lookup"><span data-stu-id="3c659-106">To correct this error, change the anonymous method to a [lambda expression](../operators/lambda-expressions.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="3c659-107">예제</span><span class="sxs-lookup"><span data-stu-id="3c659-107">Example</span></span>

<span data-ttu-id="3c659-108">다음 예제에서는 CS1946 오류가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="3c659-108">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
