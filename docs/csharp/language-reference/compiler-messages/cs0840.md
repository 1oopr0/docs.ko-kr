# <a name="compiler-error-cs0840"></a><span data-ttu-id="567b7-103">컴파일러 오류 CS0840</span><span class="sxs-lookup"><span data-stu-id="567b7-103">Compiler Error CS0840</span></span>
<span data-ttu-id="567b7-104">‘Property name’은(는) abstract 또는 extern으로 표시되어 있지 않으므로 본문을 선언해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="567b7-104">'Property name' must declare a body because it is not marked abstract or extern.</span></span> <span data-ttu-id="567b7-105">자동으로 구현된 속성은 get 및 set 접근자를 정의해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="567b7-105">Automatically implemented properties must define both get and set accessors.</span></span>  
  
 <span data-ttu-id="567b7-106">일반 속성이 `abstract` 또는 `extern`으로 표시되어 있지 않거나 `partial` 형식의 멤버이면 본문을 제공해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="567b7-106">Unless a regular property is marked as `abstract` or `extern`, or is a member of a `partial` type, it must supply a body.</span></span> <span data-ttu-id="567b7-107">자동으로 구현된 속성은 접근자 본문을 제공하지 않고 두 접근자를 모두 지정해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="567b7-107">Auto-implemented properties do not provide accessor bodies, but they must specify both accessors.</span></span> <span data-ttu-id="567b7-108">읽기 전용의 자동 구현 속성을 만들려면 set 접근자를 `private`으로 설정합니다.</span><span class="sxs-lookup"><span data-stu-id="567b7-108">To create a read-only auto-implemented property, make the set accessor `private`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="567b7-109">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="567b7-109">To correct this error</span></span>  
  
1. <span data-ttu-id="567b7-110">누락된 본문 또는 접근자를 제공하거나, 해당 항목 및/또는 바깥쪽 형식의 [abstract](../keywords/abstract.md), [extern](../keywords/extern.md) 또는 [partial(형식)](../keywords/partial-type.md) 한정자를 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="567b7-110">Supply the missing body or accessor or else use the [abstract](../keywords/abstract.md), [extern](../keywords/extern.md), or [partial (Type)](../keywords/partial-type.md) modifiers on it and/or its enclosing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="567b7-111">예제</span><span class="sxs-lookup"><span data-stu-id="567b7-111">Example</span></span>  
 <span data-ttu-id="567b7-112">다음 예제에서는 CS0840을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="567b7-112">The following example generates CS0840:</span></span>  
  
```csharp  
// cs0840.cs  
// Compile with /target:library  
using System;  
class Test  
{  
    public int myProp { get; } // CS0840  
  
    // to create a read-only property  
    // try the following line instead  
    public int myProp2 { get; private set; }  
  
}  
```
