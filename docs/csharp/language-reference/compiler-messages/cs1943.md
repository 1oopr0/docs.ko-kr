# <a name="compiler-error-cs1943"></a><span data-ttu-id="b8ac1-103">컴파일러 오류 CS1943</span><span class="sxs-lookup"><span data-stu-id="b8ac1-103">Compiler Error CS1943</span></span>
<span data-ttu-id="b8ac1-104">형식이 ‘type’인 식은 소스 형식이 ‘type’인 쿼리 식의 후속 from 절에서 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="b8ac1-104">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="b8ac1-105">‘method’ 호출 시 형식을 유추하지 못했습니다.</span><span class="sxs-lookup"><span data-stu-id="b8ac1-105">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="b8ac1-106">모든 범위 변수는 쿼리 가능 형식을 나타내야 합니다.</span><span class="sxs-lookup"><span data-stu-id="b8ac1-106">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="b8ac1-107">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="b8ac1-107">To correct this error</span></span>  
  
1. <span data-ttu-id="b8ac1-108">형식이 `IEnumerable`, `IEnumerable<T>` 또는 파생된 인터페이스를 구현하는 쿼리 가능 형식인지, 아니면 쿼리 패턴이 정의된 다른 형식인지 확인합니다.</span><span class="sxs-lookup"><span data-stu-id="b8ac1-108">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>  
  
2. <span data-ttu-id="b8ac1-109">형식이 제네릭이 아닌 `IEnumerable`이면 범위 변수에 명시적 형식을 제공합니다.</span><span class="sxs-lookup"><span data-stu-id="b8ac1-109">If the type is a non-generic `IEnumerable`, provide an explicit type on the range variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b8ac1-110">예제</span><span class="sxs-lookup"><span data-stu-id="b8ac1-110">Example</span></span>  
 <span data-ttu-id="b8ac1-111">다음 코드에서는 CS1943 오류가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="b8ac1-111">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
