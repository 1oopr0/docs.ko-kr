 <span data-ttu-id="7113a-105">인터페이스로/로부터 사용자 정의 변환을 만들 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="7113a-105">You cannot create a user-defined conversion to or from an interface.</span></span> <span data-ttu-id="7113a-106">변환 루틴이 필요한 경우 인터페이스를 클래스로 설정하거나 인터페이스에서 클래스를 파생시켜 이 오류를 해결합니다.</span><span class="sxs-lookup"><span data-stu-id="7113a-106">If you need the conversion routine, resolve this error by making the interface a class or derive a class from the interface.</span></span>  
  
 <span data-ttu-id="7113a-107">다음 샘플에서는 CS0552를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="7113a-107">The following sample generates CS0552:</span></span>  
  
```csharp  
// CS0552.cs  
public interface ii  
{  
}  
  
public class a  
{  
   // delete the routine to resolve CS0552  
   public static implicit operator ii(a aa) // CS0552  
   {  
      return new ii();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
