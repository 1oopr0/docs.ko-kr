# <a name="compiler-warning-level-1-cs1616"></a><span data-ttu-id="edbb5-103">컴파일러 경고(수준 1) CS1616</span><span class="sxs-lookup"><span data-stu-id="edbb5-103">Compiler Warning (level 1) CS1616</span></span>
<span data-ttu-id="edbb5-104">‘option’ 옵션은 소스 파일 또는 추가된 모듈에 지정된 ‘attribute’ 특성을 재정의합니다.</span><span class="sxs-lookup"><span data-stu-id="edbb5-104">Option 'option' overrides attribute 'attribute' given in a source file or added module</span></span>  
  
 <span data-ttu-id="edbb5-105">이 경고는 소스에 있는 <xref:System.Reflection.AssemblyKeyFileAttribute> 또는 <xref:System.Reflection.AssemblyKeyNameAttribute> 어셈블리 특성이 프로젝트 속성에 지정된 [-keyfile](../compiler-options/keyfile-compiler-option.md) 또는 [-keycontainer](../compiler-options/keycontainer-compiler-option.md) 명령줄 옵션 또는 키 파일 이름 또는 키 컨테이너와 충돌할 때 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="edbb5-105">This warning occurs if the assembly attributes <xref:System.Reflection.AssemblyKeyFileAttribute> or <xref:System.Reflection.AssemblyKeyNameAttribute> found in source conflict with the [-keyfile](../compiler-options/keyfile-compiler-option.md) or [-keycontainer](../compiler-options/keycontainer-compiler-option.md) command line option or key file name or key container specified in the Project Properties.</span></span>  
  
 <span data-ttu-id="edbb5-106">아래 예제에서는 이름이 `cs1616.snk`인 키 파일이 있다고 가정합니다.</span><span class="sxs-lookup"><span data-stu-id="edbb5-106">For the example below, assume you have a key file named `cs1616.snk`.</span></span> <span data-ttu-id="edbb5-107">다음 명령줄을 사용하여 이 파일을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="edbb5-107">Generate this file with the command line:</span></span>  
  
```console  
sn -k CS1616.snk  
```  
  
 <span data-ttu-id="edbb5-108">다음 샘플에서는 CS1616 경고가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="edbb5-108">The following sample generates CS1616:</span></span>  
  
```csharp  
// CS1616.cs  
// compile with: /keyfile:cs1616.snk  
using System.Reflection;  
  
// To fix the error, remove the next line  
[assembly: AssemblyKeyFile("cs1616b.snk")]  // CS1616  
  
class C  
{  
  public static void Main()  
  {  
  }  
}  
```
