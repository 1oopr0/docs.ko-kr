# <a name="compiler-error-cs1708"></a><span data-ttu-id="7f35f-103">컴파일러 오류 CS1708</span><span class="sxs-lookup"><span data-stu-id="7f35f-103">Compiler Error CS1708</span></span>

<span data-ttu-id="7f35f-104">고정 크기 버퍼는 지역 변수나 필드를 통해서만 액세스할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="7f35f-104">Fixed size buffers can only be accessed through locals or fields</span></span>  
  
 <span data-ttu-id="7f35f-105">C# 2.0의 새로운 기능은 `struct` 내부에서 인라인 배열을 정의하는 기능입니다.</span><span class="sxs-lookup"><span data-stu-id="7f35f-105">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="7f35f-106">이러한 배열은 지역 변수나 필드를 통해서만 액세스할 수 있으며, 식의 왼쪽에서 중간 값으로 참조할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="7f35f-106">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="7f35f-107">또한, `static` 또는 `readonly`인 필드를 통해 배열에 액세스할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="7f35f-107">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>  
  
 <span data-ttu-id="7f35f-108">이 오류를 해결하려면 배열 변수를 정의하고 인라인 배열을 변수에 할당합니다.</span><span class="sxs-lookup"><span data-stu-id="7f35f-108">To resolve this error, define an array variable, and assign the in-line array to the variable.</span></span> <span data-ttu-id="7f35f-109">또는 인라인 배열을 나타내는 필드에서 `static` 또는 `readonly` 한정자를 제거합니다.</span><span class="sxs-lookup"><span data-stu-id="7f35f-109">Or, remove the `static` or `readonly` modifier from the field representing the in-line array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7f35f-110">예제</span><span class="sxs-lookup"><span data-stu-id="7f35f-110">Example</span></span>  

 <span data-ttu-id="7f35f-111">다음 샘플에서는 CS1708을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="7f35f-111">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
