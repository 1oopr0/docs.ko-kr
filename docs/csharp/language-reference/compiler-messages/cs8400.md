# <a name="compiler-error-cs8400"></a>컴파일러 오류 CS8400

‘feature’ 기능은 C# 8.0에서 사용할 수 없습니다. 언어 버전 \<version> 이상을 사용하세요.

## <a name="to-correct-this-error"></a>이 오류를 해결하려면

프로젝트를 오류에 언급된 버전 이상으로 업그레이드합니다.
