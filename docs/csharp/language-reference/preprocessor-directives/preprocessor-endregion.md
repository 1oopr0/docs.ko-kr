# <a name="endregion-c-reference"></a>#endregion(C# 참조)

`#endregion`은 [#region](./preprocessor-region.md) 블록의 끝을 표시합니다. 다음은 그 예입니다.  
  
```csharp
#region MyClass definition  
class MyClass
{  
    static void Main()
    {  
    }  
}  
#endregion  
```
