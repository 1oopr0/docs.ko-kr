# <a name="ascending-c-reference"></a>ascending(C# 참조)
`ascending` 상황별 키워드는 쿼리 식의 [orderby 절](./orderby-clause.md)에서 사용되어 정렬 순서를 가장 작은 값에서 가장 큰 값의 오름차순으로 지정합니다. `ascending`은 기본 정렬 순서이므로 지정할 필요가 없습니다.
