# <a name="imetadataemitdefineuserstring-method"></a>IMetaDataEmit::DefineUserString 메서드
지정 된 리터럴 문자열에 대 한 메타 데이터 토큰을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT DefineUserString (
    [in]  LPCWSTR     szString,
    [in]  ULONG       cchString,
    [out] mdString    *pstk
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `szString`  
 진행 저장할 사용자 문자열입니다.  
  
 `cchString`  
 진행 의 와이드 문자 수입니다 `szString` .  
  
 `pstk`  
 제한이 할당 된 문자열 토큰입니다.
