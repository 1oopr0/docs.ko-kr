# <a name="using-application-protocols"></a>애플리케이션 프로토콜 사용

.NET Framework는 일반적으로 사용되는 인터넷 애플리케이션 프로토콜을 지원합니다. 이 섹션에서는 [HTTP](http.md), [TCP](using-tcp-services.md), [UDP](using-udp-services.md) 프로토콜을 사용하는 방법에 대한 정보와 [Windows 소켓](sockets.md) 인터페이스를 사용하여 사용자 지정 프로토콜을 구현하는 방법에 대한 정보가 포함되어 있습니다.
