<span data-ttu-id="c4c0d-110">샘플에서는 `Person` 데이터 계약을 사용하여 serialization 및 deserialization을 보여 줍니다.</span><span class="sxs-lookup"><span data-stu-id="c4c0d-110">The sample uses a `Person` data contract to demonstrate serialization and deserialization.</span></span>

```csharp
    [DataMember]
    internal string name;

    [DataMember]
    internal int age;
}
```

```csharp

//Set up Person object...
MemoryStream stream1 = new MemoryStream();
DataContractJsonSerializer ser = new DataContractJsonSerializer(typeof(Person));
ser.WriteObject(stream1, p);
```

 <span data-ttu-id="c4c0d-112">메모리 스트림에는 유효한 JSON 데이터가 포함되어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="c4c0d-112">The memory stream contains valid JSON data.</span></span>


 <span data-ttu-id="c4c0d-115">`p2` 개체를 확인하면 JSON 데이터가 올바르게 역직렬화된 것을 알 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="c4c0d-115">Examining the `p2` object reveals that the JSON data has been deserialized correctly.</span></span>
