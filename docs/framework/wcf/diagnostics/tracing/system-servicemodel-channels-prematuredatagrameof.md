## <a name="description"></a>Description  
 다이어그램 채널에서 null 메시지(채널의 신호 끝)가 검색되었지만 채널이 아직 열림 상태입니다. 이는 다이어그램 채널에서 오류가 발생하여 디멀티플렉서 수신 루프 종료가 중단되었음을 나타냅니다.
