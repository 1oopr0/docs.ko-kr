## <a name="examples"></a><span data-ttu-id="3a585-208">예</span><span class="sxs-lookup"><span data-stu-id="3a585-208">Examples</span></span>  
 <span data-ttu-id="3a585-209">다음 명령은 C# 코드로 **Root\cimv2** 네임스페이스의 **Win32_LogicalDisk** WMI 클래스에 대한 관리되는 클래스를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="3a585-209">The following command generates a managed class in C# code for the **Win32_LogicalDisk** WMI class in the **Root\cimv2** namespace.</span></span> <span data-ttu-id="3a585-210">도구에서 관리되는 클래스를 **ROOT.CIMV2.Win32** 네임스페이스의 c:\disk.cs에 소스 파일로 작성합니다.</span><span class="sxs-lookup"><span data-stu-id="3a585-210">The tool writes the managed class to the source file at c:\disk.cs in the **ROOT.CIMV2.Win32** namespace.</span></span>  
  
```console  
mgmtclassgen Win32_LogicalDisk /n root\cimv2 /l CS /p c:\disk.cs  
```  
  
 <span data-ttu-id="3a585-211">다음 코드 예제에서는 생성된 클래스를 프로그래밍 방식으로 사용하는 방법을 보여 줍니다.</span><span class="sxs-lookup"><span data-stu-id="3a585-211">The following code example shows how to use a generated class programmatically.</span></span> <span data-ttu-id="3a585-212">첫째, 클래스의 인스턴스가 열거되고 경로가 인쇄됩니다.</span><span class="sxs-lookup"><span data-stu-id="3a585-212">First, an instance of the class is enumerated and the path is printed.</span></span> <span data-ttu-id="3a585-213">그런 다음, 초기화할 생성된 클래스의 인스턴스는 WMI의 인스턴스를 사용하여 만들어집니다.</span><span class="sxs-lookup"><span data-stu-id="3a585-213">Next, an instance of the generated class to be initialized is created with an instance of WMI.</span></span> <span data-ttu-id="3a585-214">`Process`는 **Win32_Process**에 대해 생성된 클래스이고 `LogicalDisk`는 **Root\cimv2** 네임스페이스에서 **Win32_LogicalDisk**에 대해 생성된 클래스입니다.</span><span class="sxs-lookup"><span data-stu-id="3a585-214">`Process` is the class generated for **Win32_Process** and `LogicalDisk` is the class generated for **Win32_LogicalDisk** in the **Root\cimv2** namespace.</span></span>  
  
```vb  
Imports System  
Imports System.Management  
Imports ROOT.CIMV2.Win32  
  
Public Class App
   Public Shared Sub Main()
      ' Enumerate instances of the Win32_process.  
      ' Print the Name property of the instance.  
      Dim ps As Process
      For Each ps In  Process.GetInstances()  
         Console.WriteLine(ps.Name)  
      Next ps  
  
      ' Initialize the instance of LogicalDisk with  
      ' the WMI instance pointing to logical drive d:.  
      Dim dskD As New LogicalDisk(New _  
         ManagementPath("win32_LogicalDisk.DeviceId=""d:"""))  
      Console.WriteLine(dskD.Caption)  
   End Sub  
End Class  
```  
  
```csharp  
using System;  
using System.Management;  
using ROOT.CIMV2.Win32;  
  
public class App  
{  
   public static void Main()  
   {  
      // Enumerate instances of the Win32_process.  
      // Print the Name property of the instance.  
      foreach(Process ps in Process.GetInstances())  
      {  
         Console.WriteLine(ps.Name);  
      }  
  
      // Initialize the instance of LogicalDisk with  
      // the WMI instance pointing to logical drive d:.  
      LogicalDisk dskD = new LogicalDisk(new ManagementPath(  
        "win32_LogicalDisk.DeviceId=\"d:\""));  
      Console.WriteLine(dskD.Caption);  
   }  
}  
```
