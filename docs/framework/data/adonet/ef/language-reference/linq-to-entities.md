 <span data-ttu-id="08ebe-155">구체화는 쿼리 결과를 CLR 형식으로 클라이언트에게 다시 반환하는 프로세스입니다.</span><span class="sxs-lookup"><span data-stu-id="08ebe-155">Materialization is the process of returning query results back to the client as CLR types.</span></span> <span data-ttu-id="08ebe-156">LINQ to Entities에서는 쿼리 결과 데이터 레코드가 반환되지 않으며 사용자나 Entity Framework에 의해 정의되었거나 컴파일러에 의해 생성된(익명 형식) 기본 CLR 형식이 항상 사용됩니다.</span><span class="sxs-lookup"><span data-stu-id="08ebe-156">In LINQ to Entities, query results data records are never returned; there is always a backing CLR type, defined by the user or by the Entity Framework, or generated by the compiler (anonymous types).</span></span> <span data-ttu-id="08ebe-157">모든 개체 구체화는 Entity Framework에 의해 수행됩니다.</span><span class="sxs-lookup"><span data-stu-id="08ebe-157">All object materialization is performed by the Entity Framework.</span></span> <span data-ttu-id="08ebe-158">Entity Framework와 CLR 간의 매핑 실패로 인해 오류가 발생하면 개체 구체화 동안 예외가 throw됩니다.</span><span class="sxs-lookup"><span data-stu-id="08ebe-158">Any errors that result from an inability to map between the Entity Framework and the CLR will cause exceptions to be thrown during object materialization.</span></span>  
  
 <span data-ttu-id="08ebe-159">쿼리 결과는 대개 다음 중 하나로 반환됩니다.</span><span class="sxs-lookup"><span data-stu-id="08ebe-159">Query results are usually returned as one of the following:</span></span>  
  
- <span data-ttu-id="08ebe-160">개념적 모델에 정의된 복합 형식의 프로젝션 또는 0개 이상의 형식화된 엔터티 개체가 포함된 컬렉션</span><span class="sxs-lookup"><span data-stu-id="08ebe-160">A collection of zero or more typed entity objects or a projection of complex types defined in the conceptual model.</span></span>  
  
- <span data-ttu-id="08ebe-161">Entity Framework에서 지 원하는 CLR 형식입니다.</span><span class="sxs-lookup"><span data-stu-id="08ebe-161">CLR types that are supported by the Entity Framework.</span></span>  
  
- <span data-ttu-id="08ebe-162">인라인 컬렉션</span><span class="sxs-lookup"><span data-stu-id="08ebe-162">Inline collections.</span></span>  
  
- <span data-ttu-id="08ebe-163">익명 형식</span><span class="sxs-lookup"><span data-stu-id="08ebe-163">Anonymous types.</span></span>
