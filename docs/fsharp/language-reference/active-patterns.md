# <a name="active-patterns"></a><span data-ttu-id="8d80c-103">활성 패턴</span><span class="sxs-lookup"><span data-stu-id="8d80c-103">Active Patterns</span></span>

<span data-ttu-id="8d80c-104">*활성 패턴* 을 사용 하면 입력 데이터를 구분 하는 명명 된 파티션을 정의할 수 있으므로 구분 된 공용 구조체의 경우와 마찬가지로 패턴 일치 식에 이러한 이름을 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="8d80c-104">*Active patterns* enable you to define named partitions that subdivide input data, so that you can use these names in a pattern matching expression just as you would for a discriminated union.</span></span> <span data-ttu-id="8d80c-105">활성 패턴을 사용하여 각 파티션에 대한 사용자 지정 방식으로 데이터를 분해할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="8d80c-105">You can use active patterns to decompose data in a customized manner for each partition.</span></span>

## <a name="syntax"></a><span data-ttu-id="8d80c-106">구문</span><span class="sxs-lookup"><span data-stu-id="8d80c-106">Syntax</span></span>

```fsharp
// Active pattern of one choice.
let (|identifier|) [arguments] valueToMatch = expression

// Active Pattern with multiple choices.
// Uses a FSharp.Core.Choice<_,...,_> based on the number of case names. In F#, the limitation n <= 7 applies.
let (|identifer1|identifier2|...|) valueToMatch = expression

// Partial active pattern definition.
// Uses a FSharp.Core.option<_> to represent if the type is satisfied at the call site.
let (|identifier|_|) [arguments ] valueToMatch = expression
```
