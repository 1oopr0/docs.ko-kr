### <a name="systemnetpeertopeercollaboration-unavailable-on-windows-8"></a>Windows 8에서 System.Net.PeerToPeer.Collaboration을 사용할 수 없습니다

#### <a name="details"></a>설명

System.Net.PeerToPeer.Collaboration 네임스페이스는 Windows 8 이상에서 사용할 수 없습니다.

|:--------|:------------|

|버전|4.5|
|형식|런타임
