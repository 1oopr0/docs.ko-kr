### <a name="compiler-support-for-type-forwarding-when-multi-targeting-mscorlib"></a>mscorlib를 멀티타기팅할 때 형식 전달을 위한 컴파일러 지원

#### <a name="details"></a>설명

새로운 CodeDOM 기능을 통해 컴파일러가 .NET Framework 4.5 버전의 mscorlib.dll 대신 대상 버전의 mscorlib.dll에 대해 컴파일할 수 있습니다.

| 이름    | 값       |
|:--------|:------------|
| Scope   |Microsoft Edge|
|버전|4.5|
|형식|런타임|

#### <a name="affected-apis"></a>영향을 받는 API

API 분석을 통해 검색할 수 없습니다.
